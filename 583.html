＊男木島?男木島交流館


男木島（Image Source : Getty Creative/iStockphoto）

位於日本本州、四國和九州之間的瀨戶內海，面積超過2萬3千平方公里，海域內星羅棋布700多座島嶼，環繞內海的陸地區域通稱為瀨戶內地方。雖然過往因為地利之便成為交通動脈，不過因為全球化的衝擊，當地人口逐漸老化、流失，有著沒落的趨勢??


2013年瀨戶內國際藝術祭展出於岡山縣宇野港的大型海鯛裝置藝術，出自日本藝術團體Yodogawa Tecnique之手。（Photoby JIJI PRESS/AFP, Image Source : Getty Editorial）

至於它的旅館以及島上聚落有關藝術作品的課程，經常出現在歐美豪華度假酒店雜誌的介紹中，知名旅遊雜誌《CondéNast》就曾推薦它是《世界上七大值得一看的地方》，也因此倍樂生之家博物館在海外引起相當大的關注～～

整體建築以「男」、「木」、島」三個字的日文、漢字、英文拼音組成，播稱為「男木島之魂」的它，既是候船室也是留客服務中心，還能租借腳踏車，到了藝術祭期間，又搖身一變成為接待中心。


男木島（Photo by Zhizhao Wu, Image Source : Getty Editorial）

1992年成立的倍樂生之家博物館（ベネッセハウスミュージアム），是倍樂生藝術場直島的核心，結合美術館與旅館的它，已經成為海外知名的日本度假設施之一。從博物館和旅館建築、庭院和公共區域，到連接的森林與海灘，都是以「自然、建築和藝術」共生的理念設計。


倍樂生之家博物館中Richard Long的作品《Full Moon Stone Circle》（Image Source :Getty Editorial）

從上外下看，外觀彷彿一顆巨大的水滴，也讓人聯想起幽浮。出自三度獲得建築普立茲獎、身為SANAA建築師事務所代表的西澤立衛設計，建築頂部有兩個大洞，好讓外面的陽光與空氣進入室內，以混凝土打造的圓形結構，是一個鏤空的外殼，最高高度達4.5公尺，當中沒有任何一根支柱！


豐島美術館的咖啡館（Image Source : Getty Creative/500px Plus）

直島在1960年代開始發展觀光產業，先是設立了露營場。後來以岡山市為根據地的教育出版公司倍樂生（ベネッセコーポレーション），1987年開始搜收購島上土地，它不只致力在當地推展藝術活動，同時以打造文化村為目標，最後形成今日的「倍樂生藝術場直島」（ベネッセアートサイト直島）。


倍樂生之家博物館（Photo by Japan Stock Photo/Universal Images Group, ImageSource : Getty Editorial）

島上有間充滿原始風情的圖書館，古老的日式木頭建築中可以邊喝咖啡邊閱讀。沿著男木島的海岸線散步，還能看見《抬舟人》雕塑，幾個同心協力用頭頂舟的人只露出下半身，這件純白裝置藝術襯著藍天和大海，顯得相當醒目。島上還有一些角落還能看見隱藏版的趣味藝術。


男木島（Image Source : Getty Creative）

在高松港附近，有兩座並稱「雌雄島」的女木島（めぎじま）與男木島（おぎじま），近年來因為瀨戶內國際藝術季開辦，讓這兩座小島也逐漸成為瀨戶內海跳島旅行的目的地之一。其中特別是男木島，從高松出發大約40分鐘航程（中間會停靠女木島）。


男木島交流館（Photo by Zhizhao Wu, Image Source : Getty Editorial）

提起草間彌生，最讓人印象深刻的符號除了大大小小的圓點之外，就是南瓜。而她的黃色《南瓜》正是直島的地標。這個以纖維強化塑膠打造的立體雕塑，高2公尺、寬2.5公尺，上方布滿大大小小的黑色圓點。


草間彌生的《南瓜》（Photo by Veronique DURRUTY/Gamma-Rapho, Image Source :Getty Editorial）

出自安藤忠雄之手的「古怪」建築，和島上的景觀與歷史，展現出充滿戲劇性的拉力。點綴其中的藝術品，不只考量美術館的結構，更結合瀨戶內海的風景和村莊的歷史，可說是專為直島量身打造，特別是近年來館內採購的作品，都是「現場特定作品」，也就是說針對特定地點創作的藝術品！


陳列於倍樂生之家博物館庭園裡的雕塑（(Photo by Eric Lafforgue/Art In All Of Us/Corbis,Image Source : Getty Editorial)

豐島區有座清水神社，傳說因為弘法大師在唐櫃岡挖出了一口湧泉而得名，在這座神社附近，有件由來自東京的雕刻家兼版畫家青木野枝創作的《空之粒子》。雖然採用看起來很重的鐵，不過卻呈現出輕盈且流暢的感覺，一個個朝天空延伸的圓圈，彷彿在空中跳舞的粒子，讓人視線忍不住隨之往上，仰望天空～～

除此之外，男木島還是一座貓島，島上沒有天敵的貓咪過著悠閒的生活，隨意在巷弄間逛來晃去，或是直接躺在路上曬太陽，讓人實在很羨慕牠們的愜意～～

面積只有1.34平方公里的男木島，人口不到200人。一抵達當地港口，就能看見一棟純白色的建築，造型獨特的它，上方罩著鏤空的雕刻裝飾，彷彿一件裝置藝術，這裏是男木島交流館。


男木島交流館（Photo by Zhizhao Wu, Image Source : Getty Editorial）

這位日本前衛反形式主義「物派」大師，使用石頭、木頭、鐵板等媒材，以點和線唯二元素進行創作，近年來在國際間擁有相當高的評價。矗立於一座安靜的山谷中，安藤忠雄利用當地地景，打造了一座面積將近1萬平方公尺、半地下的鋼筋混凝土結構。


李禹煥美術館（Photo by Eric Lafforgue/Art In All Of Us/Corbis, ImageSource : Getty Editorial）

美術館中只有一件作品《母型／Matrix》，出自日本藝術家內藤禮的之手，她擅長將安靜且精緻的雕塑，結合具有緊張感的空間，而豐島美術館正是建築與藝術作品融為一體的最佳闡述，隨著周圍環境與時間流逝而產生變化??除了美術館以外，這裡還有藏身於方形洞穴中的接待處，以及白色半圓形的咖啡館。後者同樣出自西澤立衛之手，在這個依舊沒有任何梁柱的拱形空間裡，開著一扇透光的天窗。

豐島（Image Source : Getty Creative）

＊直島?地中美術館


地中美術館（Image Source : Getty Creative）

附屬於香川縣的直島（なおしま），位於高松市北方約13公里處，從高松港搭乘遊輪前往大約50分鐘。崇德上皇曾經在12世紀的保元之亂後、流放讚岐途中在此停留，為它取了今日的名字。


草間彌生的《南瓜》（Photo by Veronique DURRUTY/Gamma-Rapho, Image Source :Getty Editorial）

從正面看，除了6公尺高、50公尺長的混凝土牆外，什麼都看不見。在這處從山谷通往大海的半開放式空間裡，三角形（廣場）與長方形（長牆）的搭配設計，巧妙融合自然，產生充滿節奏的空間，正面的六角混凝土柱——《李氏之柱》，拉開了垂直與水平線條之間的張力，帶來某種緊張卻又平衡的力量??

配合2010年瀨戶內國際藝術節開幕的李禹煥美術館（リ?ウーファンびじゅつかん），也由福武財團營運、安藤忠雄設計。它是1936年出生於韓國、以「極簡美學」著稱的畫家兼雕塑家李禹煥的首間私人美術館，館內陳列著這位藝術家1970年代至今的多樣作品。


李禹煥美術館（Photo by Eric Lafforgue/Art In All Of Us/Corbis, ImageSource : Getty Editorial）

＊豐島?青木野枝《空之粒子》


《空之粒子》（Image Source : Getty Creative）

位於直島和小豆島之間，周長20公里、面積只有14.5平方公里的豐島（てしま），人口大約1,000多人，過去長期以來都飼育乳牛，因此被稱為「牛奶之島」，島上壇山擁有豐富湧泉，孕育著農作，因此可以欣賞到連綿不絕的梯田風光。它和直島、高松、小豆島等地都有船隻往返，從高松港出發大約35分鐘。


豐島美術館（Photo by Eric Lafforgue/Art In All Of Us/Corbis, Image Source: Getty Editorial）

地中美術館只永久展出三位藝術家的作品，他們分別是印象派大師莫內、美國地景極限藝術家沃爾特?德?瑪利亞（Walter deMaria），以及擅長以光線和空間為創作素材的美國當代藝術家詹姆斯?特瑞爾（JamesTurrell）。安藤忠雄特別為他們的作品打造了獨一無二的空間，不但充分發揮了藝術家的想法，同時也讓這座博物館成為其他地方再也找不到的場地限定美術館！

於是為了活化當地，並且讓這些島嶼的特色再度被看見，瀨戶內國際藝術祭就在這樣的背景下誕生。舉辦地點橫跨香川縣和岡山縣，主要舉辦島嶼包括直島、豐島、男木島、小豆島等，現在就讓我們展開一趟跳島旅行，看看有哪些亮點吧！

你聽說過瀨戶內國際藝術祭嗎？這個2010年開始舉辦的活動，以各個島嶼為舞台，結合當地傳統文化、美麗自然景觀以及當代藝術，如今已經成為日本國內最大的藝術盛事之一！今年（2022年）正好是第五屆，該藝術季分為春、夏、秋三季，許多人之前因為疫情的關係無法前往，不過目前秋季活動正在舉行中，預計展覽到11月6日。即使趕不及在藝術季時拜訪，主要會場的島嶼還是有固定的裝置藝術或美術館，無論何時都能前往參觀，展開迷人的跳島旅程～～（日本目前入境防疫觀光政策，取消入境人數限制，同時開放海外旅客自由行和短期免簽證旅客入境，台灣人可以以三劑疫苗証明，或72小時內PCR陰性證明，免除日本入境檢疫隔離。）


直島與瀨戶內海（Image Source : Getty Creative）

＊直島?倍樂生之家博物館


倍樂生之家博物館（Image Source : Getty Creative）

2004年時，以倍樂生公司名譽顧問福武宗一郎為主席的直島福五美術館團（今日的福武財團）成立，它在直島南部山上呈梯田狀的立體鹽場遺址地下室，興建了整體建築都「埋」在地下的地中美術館（ちちゅうびじゅつかん），從地面上，只能看見方形和幾何狀的開口。


地中美術館（Photo by Iain Masterton/Construction Photography/Avalon,Image Source : Getty Editorial）

這棟建築同樣出自安藤忠雄的設計，即使位於地下，依舊擁有自然採光，其中一大亮點，在於美術館中展出的藝術品，會因為一天之中的時間變化，產生不同的面貌，給人整座建築就是一件巨大作品的印象～～


地中美術館（Photo by Iain Masterton/Construction Photography/Avalon,Image Source : Getty Editorial）

這件作品是她1994年時的創作，並且曾在「Open Air \”94 \”Out of Bounds\”—海景之中的現代美術展—」中展出。儘管她創作了許多以南瓜為主題的作品，不過它是直島的首座戶外雕塑，如今在直島一座深入大海的堤道上永久展出（除了颱風來臨時會暫時撤離），歡迎著前來的遊客！

近年來，因為豐島美術館（てしまびじゅつかん）的出現，以及島上多處現代藝術展示，讓它搖身一變成為頗具知名度的藝術之島。豐島美術館搭上2010年瀨戶內國際藝術展的專車，10月17日正式對外開放，在豐島東北部俯瞰著瀨戶內海的丘陵，迎接著遊客的到來～～


豐島美術館（Photo by Francois GOUDIER/Gamma-Rapho, Image Source : GettyEditorial）

美術館主要的展示品包括安藤忠雄的倍樂生之家模型、大竹伸朗的《造船廠工程：切弓》在內的多件作品、安迪?沃荷的《花卉》、蔡國強的《文化大混浴 直島項目磁磚圖》、博羅夫斯基（JonathanBorofsky）的《三個喋喋不休的人》等，之前提到過的《南瓜》當然也是它的收藏之一。

前沿拓展：岡山縣,南瓜

去惯了京都、大阪和东京这些都会地区？想改走文艺路线的日本之旅？那你一定会喜欢上这里——濑户内海地区。

濑户内海区域指的是夹在本岛的冈山县和四国地方的香川县中间，在这片海域上散落着数十个小岛。这里有着碧海蓝天的海岛生活，走的却不是奢华的度假风，而是一种远离世俗，教人放下一切烦恼的，被艺术品和美术馆环绕的慢生活。

这里最出名的是每3年举办一次的“濑户内国际艺术节”，最近一次将在2019年春季展开。艺术节期间岛上将会开设活动和进驻非常设的艺术品，往来各个岛屿间的船只也会增加班次，大家为了收集艺术节护照上面的作品印章而四处奔波也是一种乐趣，但如果你想更轻松的享受濑户内海独特的生活氛围，我们更推荐你在艺术节以外的时间到访。

接下来要介绍的，

都是各个岛上的常设艺术作品。

直 岛

直岛是艺术大师的云集之地，一个岛上就拥有草间弥生、安藤忠雄、莫奈、妮基·桑法勒等超多艺术家的作品，甚至有许多西方游客来日本就只为了来到直岛。在直岛上最推荐的交通方式是在下船后就在港口边租单车，迎着海风出发寻找直岛为你准备的艺术惊喜。

地中美术馆

▼

地中美术馆的设计理念是“考虑了自然与人类的场所”，2004年时设立的。为了尽可能不改变直岛原来的风景，将建筑的大半部分都埋于地下。美术馆的建筑由安藤忠雄设计，常态展出莫奈、詹姆斯·特瑞尔及沃尔特·德·玛利亚的作品。透过天井透进来的光线变化让地中美术馆时刻都有不一样的景观，建物本身就是一个艺术品。

Benesse House 美术馆

▼

位于岛上的高处，可以远眺直岛的风景，除了室内的收藏，室外的草地上也有许多妮基·桑法勒的作品。走下山坡，就会看见最有名的草间弥生黄色南瓜。

家计划（家プロジェクト）

▼

直岛家计划改造许多岛上的民房，并在屋内设有不同艺术家的作品，共包含“角屋”、“南寺”、“金座”、“护王神社”、“石桥”、“碁会所”、“牙医诊所”七个地方，散落在本村港区域，之间的距离徒步皆可到达。

其中“南寺”是最热门的一个，每日参观人数有限，建议大家可以优先安排，里面的“体验”让人印象深刻，这里就不提前剧透了，一定要亲身体验。

直岛钱汤“I??汤”

▼

逛了一天的直岛艺术品，晚上还个独特的享受。直岛钱汤由艺术家大竹伸朗将当地唯一的澡堂改造，从墙壁到水龙头全部重新设计，一踏进去会有种走错棚了的感觉，但又忠实的呈现了日本独有的澡堂文化。

丰 岛

来到丰岛，日语能力许可的话相当推荐住在岛上的民家民宿，直接住进居民的家中，体验真正的岛民生活。丰岛上的艺术站点较为分散，而且岛上的地形起伏多，建议租单车的旅客要租电动单车比较适合。

丰岛美术馆

▼

记得我身边第一个去过濑户内海的朋友回来后，所有美术馆里最推荐的就是丰岛美术馆，问他里面有什么，又一直神神密密的不肯说，后来亲身去过之后才知道为什么。丰岛美术馆内只有一个作品，当你走进美术馆，也就融进了作品之中，想要了解它，唯有自己去感受。

心跳保存所（心臓音のアーカイブ）

▼

在岛上的一处海边，心跳保存所收集了来自世界各地不同人的心跳声，走进黑暗的回廊里，真切地听见他人的“心声”。在这里，你可以自由浏览别人的心跳，也可以把自己的心跳声录制成光盘永久保留。

丰岛横尾馆

▼

家浦港附近的民宅改建而成的横尾忠则美术馆，建筑由永山祐子设计，分成“母屋”、“仓”、“纳屋”三个部分，作品不限于室内的平面展示，而是扩散在整个空间内。

犬 岛

相较于小豆岛、直岛和丰岛，犬岛的名气可能没有这么大，往返的船班也不多，但却完全不减这里的可看性。犬岛上的艺术作品集中在港口附近，脚程快的话3个小时左右就可以逛完全岛。

犬岛精炼所美术馆

▼

犬岛美术馆是岛上最大的建筑，过去曾是炼铜工厂，经过整修与改装在2008年以美术馆之姿再现于岛上。

家计划（家プロジェクト）

▼

犬岛上面的家计划同样有七个作品，皆在犬岛精炼所美术馆徒步范围内。走在犬岛的小路上找这几个“家”，看着田园和大海，宁静又舒心地度过一个下午。

男木岛和女木岛

想要到男木岛，要先做好不断爬坡的心理准备。岛上的道路狭小，掺杂着不断上下的坡道，却处处藏着美好。

而且男木岛还是个隐藏的猫岛。

女木岛则是因为桃太郎的故事里的“鬼岛”而成名，下了港口后，坐上公车就可以直达“鬼之洞窟”的入口。鬼之洞窟位于岛上的最高点，所以参观完之后，就可以顺便走到眺望台的位置，这里可以看见无死角的濑户内海美景。海上岛屿交错的身影，称得上是濑户内海绝景了。

还记得前面提到的直岛钱汤吗？同一位艺术家大竹伸朗的作品在女木岛上同样可以找到。将整个小学校园打造成超现实的太空基地模样，融合了原有的地景和现代艺术。（非艺术季的期间只在周末开放。）

旅行贴心tips：

·本次介绍的作品都是常设于各个岛上的，非艺术季期间能够尽情地游玩。各展览点固定休息日可以在网上查询到。

·想一口气看完这么多艺术品，建议预留5-7天的时间在濑户内海区域游玩。除了直岛和丰岛外，其他岛上的住宿选择较少，建议可以住在高松港周围的酒店再搭乘交通船前往。

每天乘着船去到一个新的岛,充满了冒险的感觉，再加上满满的艺术气息，濑户内海成为了不少文艺青年心里永远的庇护所，想去濑户内海追艺术？不用再等到2019，现在就可以出发！
