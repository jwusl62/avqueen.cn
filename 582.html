在日本東北地區旅遊時，使用「JR東日本鐵路周遊券（東北地區）」就可以用最划算的價格暢遊東北。大人日幣20,000元、兒童日幣10,000元，可以在連續的5天中在指定的區域裡無限次自由搭乘新幹線?特急列車的普通車指定席。「愉快列車」（のってたのしい列車）的系列觀光列車也可以搭乘喔！

日本已在10月中旬開放了自由行觀光，為了迎接觀光客到來，日本各界做足了準備。而在日本東北地區旅遊時，許多遊客多會利用「JR東日本鐵路周遊券」以最划算的價格暢遊東北，一直以來都受到喜歡，而JR東日本近年推出了「愉快列車」（のってたのしい列車）系列觀光列車，每輛列車都有其獨特的主題。乘著列車深入日本東部各地的同時，除了賞沿途美景、享用特色鐵路便當以外，甚至還有品酒、觀星等等為，是一場極致文化體驗，民眾前往日本旅遊時，不妨前往體驗。

乘著銀河列車觀星去！這輛「HIGH RAIL1375」觀光列車以「離天空最近的列車」為概念設計，打造獨特的觀星體驗。列車的外觀與內裝運用了大量的「星空」元素，車廂內還設有可以閱讀天文相關書籍的「HIGHRAIL展廊」，上方有星空的投影。

行駛地區：新潟縣
行駛路線：上越妙高站?十日町站、上越妙高站?越後湯澤站、上越妙高站?新潟站

喜愛日本酒的人絕不能錯過這台「越乃Shu＊Kura」觀光列車！「越乃Shu＊Kura」以新潟縣盛產的「日本酒」為主題，除了提供乘客列車專屬的日本酒、新潟在地食材製成的鐵路便當，以及在地酒免費試喝等等，車廂內還設置了特殊的「酒吧車廂」供乘客品酒。列車從2014年首次推出以來就大受歡迎，並且菜單及車上活動也持續推陳出新，深受各國旅客喜愛。

◎.【越乃Shu＊Kura】

「HIGH RAIL1375」行駛於連接山梨縣與長野縣的小海線上，會經過所有JR鐵路之中標高的最高點1,375公尺，這也是本列車「HIGH RAIL1375」的命名由來。列車在被天文學者評為「日本最漂亮的星空best3」的長野縣南牧村的「野邊山站」會有約50分鐘的停靠時間，並會在此舉行「星空觀察會」，由列車的星空介紹人帶領乘客欣賞星空，講解觀星知識。喜歡觀星的人，絕不能錯過這台神秘又獨特的觀光列車。

SQL约束的定义

SQL约束用于规定表中的数据规则。如果存在违反约束的数据行为，行为会被约束终止。

约束可以在创建表时规定（通过 CREATE TABLE语句），或者在表创建之后规定（通过 ALTER TABLE语句）。

SQL DEFAULT 约束

DEFAULT约束用于向列中插入默认值。

如果没有规定其他的值，那么会将默认值添加到所有的新记录。

一）CREATE TABLE 时的 SQL DEFAULT约束

数据库实例

CREATE TABLE student

(

IID int NOT NULL,

name varchar(255) NOTNULL,

sex varchar(255),

age varchar(255),

country varchar(255) DEFAULT\’CN\’

)

通过使用类似 GETDATE() 这样的函数，DEFAULT约束也可以用于插入系统值：

数据库实例

CREATE TABLE orders

(

OrderIdvarchar(255),

OrderNovarchar(255),

P_Id varchar(255),

OrderDate date DEFAULTGETDATE()

)

二）ALTER TABLE 时的 SQL DEFAULT 约束

数据库实例

ALTER TABLE student

ADD country SET DEAFAULT\’CN\’;

三）撤销 DEFAULT 约束

数据库实例

ALTER TABLE student

ALTER country DROPDEFAULT;
