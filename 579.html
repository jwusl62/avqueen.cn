店內的「極盛二段丼」以大份量的浮誇視覺效果，讓圖片一PO上臉書就造成極大的轟動，店內新鮮甘甜的超豐盛海鮮丼飯怎麼吃都覺得給它超級過癮的。

餐廳資訊+菜單

店內一樓是以爐端燒的用餐方式，也就是客人圍著吧檯來用餐，日料師傅會使用船槳將食物傳遞給客人。這種用餐方式是源自日本天皇到民間時，最後演變成為一個地方的特殊用餐模式。

眼前的這幅景象也實在太浮誇了，這一碗雙層丼飯飽滿到讓我們咋舌，還好剛剛真的沒再多點，所謂的二段丼指的就是二層，先看看這上面這層是豐富的刺身盤，紅、黃、白是基本款，不但新鮮甘甜、每一塊的大小及厚度都很適宜，除了魚肉刺身還有章魚、花枝、星鰻、烤蝦、鮭魚卵、海膽超多的美味，而下面這層居然還有滿滿的驚喜唷XD！

二樓是很道地的日式居酒屋模式，點好餐點會由服務人員送上二樓，走進店裡不論在一樓或二樓都有一秒到日本的錯覺，不過很有趣的是二樓的牆面佈置了許多地名，據說整個店面都是老闆自己親自設計裝潢的！這種技能也實在是太厲害啦XD！

前沿拓展：雙層丼飯,找美食

煲仔饭看似是很简单，实则要想做的美味还是讲究方法与技巧，首先制作煲仔饭选用的大米为油粘米，油粘米的做出来的米饭香滑浓香，吃起来会有浓浓的米香味。其次制作煲仔饭最为重要的一道调味料就是酱油，煲仔饭用到的酱油都是特别调配，调配出来的酱油鲜味浓郁，咸味偏低加入到煲仔饭中才更能衬托出其中的美味。下面附酱油配方

调料：冰糖50克，鸡粉10克，味精8克，味极鲜30克，冰糖老抽200克，生抽200克，清水500克。

香料：桂皮3克，草果3克，良姜1克，香叶1克，干姜1克，白芷1克，甘草2克，丁香3克，白胡椒5克，大红袍3克，小茴香6克，黄栀子1克。

做法：

洋葱切成丝，干香菇温水泡发20分钟，把香菇泡软，老腊肉撕成小块。香料洗净，用开水加白酒浸泡10分钟。将所有调料混合搅拌均匀。起锅烧油，加入50克色拉油，油温7成热下入配菜，小火炒出清香的味道，然后下入挤干水分的香菇，香菇的作用也是为了增加煲仔酱油的香气！下入腊肉，炒至锅中所有的配菜变色后，然后下入准备好的香料，下入香料炒1分钟左右，待香料颜色焦黄即可。下入搅拌好的调料，小火熬制。熬制调料不是为了增加香气，而是为了让所有的调料融化融合在一起。小火熬制2分钟，锅边冒泡微开即可关火。筛网过滤掉酱油中所有的残渣。将过滤好的酱汁回锅勾芡至浓稠，无需等烧开即可盛出。注意：加入冰糖和冰糖老抽的调料汁是属于高糖的，熬制的过程一定要用小火慢慢的熬制，大火容易变糊发苦。过滤掉的残渣用手勺挤压出剩余的酱汁。

近期，日本的夏日祭、烟火大会特别火热，可谓是刷爆我的朋友圈。但相较于日本的其他传统，我更喜欢研究他们“吃”这件事。

好比前段时间的“土用丑日”，就是日本人吃鳗鱼饭的日子。而说到鳗鱼饭，它的起源则是从江户时代开始，可谓是历史悠久。

而鳗鱼则分为关西和关东两种做法，日本关东的鳗鱼一般是从背部入手料理，而且因为本身关东地区的鳗鱼会更加肥厚，所以厨师会选择先烤、再蒸、再烤来去除腥味和油脂。

而关西的做法会相较于简单一点，一般是从腹部下刀入手，料理时会将鳗鱼用铁签串起直接烤制。然后在过程中根据需求刷酱汁，做成蒲烧鳗鱼。

夏天的味道

曾经有美食家说过，全国最好吃的日料店就在上海，而要说到鳗鱼饭我觉得也不例外。

在魔都上海，各类日料店里面基本上都少不了鳗鱼饭，除此之外还有很多专注于把一碗鳗鱼饭做到极致的店。许多店更是把关西与关东两种做法融合。今天B姐特地为大家整理了4家店，或是名店或是我觉得好吃的，但绝对都是各有长处，值得一试。

· SHARI ·

// 米饭可能比鳗鱼更好吃 //

其实我知道SHARI是因为它家的米饭，没错，就是一碗白饭让我对这家店记忆深刻。所以这次寻找鳗鱼饭的旅程中，我也特地来它家试了一下。

因为SHARI的厨房和鱼缸都是开放式的，所以B姐特地探秘了下。绿背白肚，是野生鳗鱼无疑了，SHARI的总厨介绍，这里选用的还都是1-1.2公斤的野生大鳗鱼，肉质更厚的同时，脂肪的品质也越上乘。

正好去的当天还遇到总厨在花园里烤鳗鱼，备长炭的香气真是电烤无法比拟的。这样烤出来的鳗鱼受热均匀，香气更足。

而且SHARI的总厨也是真的很随和，是一个地地道道的日本人，看我在旁边“偷师”就直接和我讲解起来了。

这里的鳗鱼就是结合了关西和关东两种做法，活鱼会先放进冰水里麻痹，然后抽神经，这样会呈现出一种“鱼死肉活”的状态。

然后鱼肉还需要低温熟成4-5天，先烤再蒸再烤，这样的鳗鱼会肉嫩皮脆，而且能更好的释放氨基酸。

上桌的时候摆盘有惊艳到我，虽说是套餐，但它不同于一般的食盒，追求精致的小伙伴可以来试试，这里一定能符合你的“审美”。

配的米饭也是它家十分出名的名产锅烧饭，使用的是总厨精心挑选混合的【SHARI】米，就是我前文所提到的。

这份饭的常规吃法也是鳗鱼3吃，1吃鳗鱼丼、2吃炭烧鳗鱼，满满地浇上自己喜欢的佐料、3吃鳗鱼茶泡饭，汤是用木鱼花绿茶出汁，十分鲜甜。

在这里品尝的每一口都超级满足，如果你不仅仅追求口腹之欲，更向往精致和本味，那SHARI是个好选择。

除了套餐外这里还有一份锅烧鳗鱼饭也是不能错过的，蒲烧鳗鱼和生米一起烹饪，酱汁满满的融合在米粒中。

上面还铺着一层白烧鳗鱼，一次吃到2种口味，这味道会让你对鳗鱼饭有一种全新对认识。

地址：永嘉路630号

电话：021-54660320

营业时间：11:30-23:00

· 鳗 重 ·

// 8个座位的小店，只有卖鳗鱼饭 //

B姐推荐的第二家，就不如SHARI那么店大了。他是一间只可容纳8人的小店，并且店里只卖鳗鱼饭。

经营这家店的是一对上海老夫妻，叔叔负责烤鳗鱼，阿姨负责前台招呼客人，叔叔曾经师从一位专做日料的日本师傅。

鳗重装鳗鱼饭的食盒也是十分讲究，都是老板从日本带回来的，当阿姨把食盒摆在你面前的时候，那感觉恍若到了日本。打开盖子就可以看到铺的满满的鳗鱼，米饭被遮的严严实实。

而且你千万别小看他家这个鳗鱼，这可是每天早上5点在店里活杀的，然后再将鳗鱼进行清洗，洗净之后先放入烤箱中进行烘烤定型，然后再用蒸汽进行蒸煮，这样可以让鳗鱼更加的绵软多汁。

鳗重的烤鳗鱼都是需要刷三次烤三次的，这样酱汁的味道才会渗透进鳗鱼里，更加入味。再次刷酱汁的鳗鱼在烤制之后，会散发出诱人的香味，色泽看起来更是诱人~

这里还有个与其他店不同的地方就是，在鳗鱼饭端上桌前，还会最后再刷上一层酱汁，这样做鳗鱼饭的味道会吃起来更加浓郁。

这家店的名字就是它家鳗鱼饭的名称，也是很有寓意的。在日本只有最高规格的鳗鱼饭才能被称为“鳗重”，普通规格的只能叫“鳗丼”。

鳗鱼表层烤到微焦，香气四溢。吃起来焦脆和软嫩相结合，口感上也更加偏甜，十分下饭！

鳗鱼下面的米饭也是非常讲究，选用的都是日本小町的大米。事先浸泡2-3小时后，再慢火煮制而成。阿姨还告诉我们，地道的吃法是先用筷子将鳗鱼切分好之后，然后用筷子将鳗鱼就着米饭一起吃。

如果你很贪心的提前把鳗鱼给吃完了，只剩下了白米饭，千万别浪费了，你还可以叫阿姨给你烤鳗酱汁淋在剩下的白米饭上。一碗超级美味的酱汁拌饭让你不想浪费一粒米！

如果你喜欢日式小店的氛围，那鳗重绝对是一个小选择。品质优秀、价格也较为合适，重要的是，在这里吃饭很有日本的氛围，简单的几个座位，大家一起吃吃饭、聊聊天，十分惬意。

地址：愚园路580号

电话：021-52689958

营业时间：11:00-14:00，17:00-21:00

· 名代宇奈备长炭烧鳗鱼饭专门店 ·

// 日本国民鳗鱼，开业当天大排场龙 //

今天会推荐这家店原因是，它是日本国民鳗鱼连锁店，在东京和大阪都有分店。刚来魔都的那会可谓是风靡到不行。

因为地处人广来福士，商场还特地为他家排队的人流指引了位置，可谓之红极一时，所以导致B姐一直望而却步。这次特地等风头过了，来看看它家到底口味。

来到这里那肯定要尝试下它家的人气产品：鳗鱼重-特上，上桌的瞬间连盖子都没有，我个人理解是因为鳗鱼量太大，盖不上….

不过这个视觉冲击是真的强，比起另外3家，它的造型更能让我有记忆点。

这里采用的做法是一蒸三烤，但表层也是刷足了酱料，保证食客每一口都能品尝到鳗鱼的香气和肉质。

香甜的秘制酱汁和鱼肉一起送入口中，每一口都是渗入灵魂的愉悦啊！但这家也有个“小问题”，就是还有少量的鱼刺，吃起来需要比较小心点。

因为这里的鳗鱼饭选择还蛮多的，所以我有另外点一份双层鳗鱼丼，分量上面更加适合一个人吃。

这家店还有温泉蛋可以单点，喜欢的话可以加点一个哦！蛋液混合着鳗鱼汁吃也是很满足的~

地址：西藏中路268号来福士广场4楼04-01D

电话：021-63301728

营业时间：11:00-22:00

· 丸 忠 ·

// 某DP前三名，上海名店 //

在魔都近千家鳗鱼饭中，丸忠可以算是技术流，内部对厨师匠人有着严格对认定，强调制作鳗鱼饭对传统技术与知识，只有获得认定的厨师匠人所制作的鳗鱼才能上餐桌。

简单来说，丸忠对于每一份上桌的鳗鱼饭都严格要求，精益求精！

本次我首先选择的是丸忠鳗鱼膳饭定食，造型上没有另外3家的华丽和惊艳，整体较为平淡。

不过鳗鱼膳饭本身就是把鳗鱼蒲烧切碎铺在米饭上，是日本爱知县有名的一道乡土料理，而且量也十分可观，下面的米饭被遮的严严实实。

丸忠的鳗鱼膳饭定食也是采用一鱼三吃的方式，第一碗吃原味，品尝鳗鱼本身的鲜美。

第二碗去腻，可以加入事先准备好的香葱、芥末、海苔等混合搅拌。

第三碗吃鲜香，就是在第二碗的基础上加入旁边的茶汤，就是一碗茶泡饭了。

整体来说丸忠的口味比较传统，而且胜在量多，如果在附近的小伙伴想吃鳗鱼饭还是可以试一试的。

同样我还点了一份这里的鳗鱼盖浇饭，鱼肉切的比较大块，酱汁也会更多点，喜欢蒲烧味重的可以尝试点下这款。

地址：世纪大道100号环球金融中心B1

电话：021-62338970

营业时间：11:00-14:30，17:30-21:00

当然，作为魔都要想找到一家好吃的鳗鱼饭肯定还有更多的选择，除了这四家如果你有好吃的店也可以在留言里告诉我，让我们一起在肥胖这条路上尽情奔跑吧！
