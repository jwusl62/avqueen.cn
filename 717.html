都歷遊客中心就在超有名的都歷沙灘天空之鏡附近，在天空之鏡拍完美美的網美照後，可以順遊到遊客中心吹吹風、望望海、稍做休息。都歷遊客中心有著廣大的綠色草地，正面對著湛藍的大海，背山面海、海天一線的美景令人心曠神怡，不自覺的放鬆了下來。與其他台東景點一樣，這裡也有不少利用漂流木做成的裝置藝術，充滿意境的藝術品，與台東的人文特色相得益彰，令人印象深刻。

地址：台東縣成功鎮新村路25號(約台11線124.5k處)

台11線景點｜都蘭觀海平台

樟原長老教會原本因為颱風被損毀，後來以帆船造型為主題整修，用白色的木板去建造，船身上還畫有船錨與海浪，仿佛聖經中「諾亞方舟」的情景在台東長濱鄉再現。樟原教會共有三層樓高，是當地原住民的聚會場所，平時是不開放入內的。從教會的側面望去，可以看到不遠處的海洋翻騰著，真的很有大船入港的感覺呢。

地址：臺東縣長濱鄉長濱鄉67-1號(約台11線75k處)

台11線景點｜八仙洞

慢點海景民宿有著米白色的外牆，門前還種著幾棵椰子樹，頗有東南亞異國風情，民宿正門口隔條馬路便是太平洋，不只從房裡便能眺望無垠的海景，從浴室裡也能看到美麗的大海，讓你邊泡澡邊看海。隔天不妨設個鬧鐘早點起床，躺在床上透過落地窗欣賞美麗的日出，自海的那一頭緩緩升起，展開美好又悠閒的旅程。

地址：花蓮縣花蓮市海濱街68號

Sea Road海道民宿

擁有180度無死角海景的盪鞦韆就在都蘭觀海平台，這裡除了超人氣的盪鞦韆，還設置了一些裝置藝術，利用漂流木製作的搖搖馬、愛心相框等，都是拍照打卡的好素材，再搭配上海天一色的絕美風景，絕對能為你的都蘭一日遊留下不少美照。

地址：台東縣東河鄉都蘭派出所對面巷子進入

台11線景點｜富山護魚區

富山護魚區是一個天然的自然保護區，早年因為居民捕撈過盛，而造成生態不平衡，後來透過禁止捕漁的政策，如今成為生態豐富、環境優美的觀光景點。來到此處，你可以親自到海邊欣賞多樣化的海洋生物，魚、蝦、蟹等小動物低頭就隨處可見，相當適合帶著孩童來一場生態體驗。

地址：台東縣卑南鄉杉原20號(約台11線約161k處)

台11線景點｜東海岸海景民宿推薦

OwlStay故事所鹽花 

地址：花蓮縣豐濱鄉 (約台11線64.5K處)

台11線景點｜樟原長老教會

豐濱新社有著遼闊的水稻田，和一望無際的藍色大海，隨著稻米的成長階段不同，插秧期的綠色稻田、成熟期的金色稻浪、休耕期的銀色水面，不論是春、夏、秋、冬何時前來，都有著不同樣貌的美景可以欣賞。新社同時也是葛瑪蘭族的聚居地，藝術家們利用竹、藤、稻等天然材料，在梯田中央建造了象徵原住民家屋的藝術作品，為這片美麗的景色，增添了些許人文氣息。

地址：花蓮縣豐濱鄉新社部落 (約台11線43.5k處)

台11線景點｜石門班哨角遊憩區

位於花蓮豐濱鄉的「月洞」又稱為「月亮之井」，洞裡的泉水沿著石灰岩壁流下，日積月累形成美麗的鐘乳石洞景觀，最特別的是，這裡並沒有進出水口，但洞內的水位卻仍會隨著月亮盈虧而有高低分別。想要進入月洞必須先走過一段木棧階梯，接著登上高處的觀海平台，這裡可以看到毫無遮蔽的海景，接著便可搭船前往神秘的月洞

奇觀囉。進到洞裡別忘了抬頭看看住在洞裡的居民─蝙蝠，這裡至少有6種以上的蝙蝠種類，密密麻麻的掛在岩洞頂端，乍看之下相當狀觀。

八仙洞曾是台灣長濱文化的起源處，由於板塊運動露出海面，再加上長時間的風蝕和風化，而形成了大大小小的許多洞窟，因為特殊且神秘的氛圍，在早期曾有許多神壇設立在此處，因此被稱為八仙洞。八仙洞的造型景觀各不相同，有最大的「靈岩洞」、最高的「崑崙洞」、「乾元洞」、「水濂洞」等多個洞穴，其中還有一個長得像台灣形狀的洞穴，旅客們不如試著找找看吧。在八仙洞旅客中心內，可以看到石器文化的文物、模型展式，是個育教於樂，同時兼顧風景與人文的景點。

地址：台東縣長濱鄉三間村水母丁1-4號(約台11線77k處)

台11線景點｜金剛大道

台11線上美麗的海景無敵多，石門班哨角遊憩區是其中一個錯過可惜的景點。開闊平緩的綠地，正面對著180度無死角的太平洋，海水顏色隨著陽光照射的強度不同，呈現出深淺不一的藍綠色，伴著陣陣帶著鹹味的海風，充滿著濃濃的夏天風情。石門班哨角遊憩區景色雖美，但景區不大，可以安排與附近熱門景點「麻糬洞」一同順遊。

地址：花蓮縣豐濱鄉(約台11線60.5k處)推薦閱讀：石門麻糬洞

台11線景點｜石梯坪

比西里岸來自於阿美族語Pisirian，原意為「養羊的地方」，這裡原只是個靠海的小村落，如今因為社區再造，而再度成為被遊客青睞的景點，比西里岸最有名的便是在海堤邊，利用漂流木製作而成的羊群們，每個到訪的遊客必定與牠拍過照才算真正的到此一遊，這裡也有11處幾米的畫作，跟著找尋這些畫作的腳步，同時體驗比西里岸幽靜淳樸的小村風光。

地址：台東縣成功鎮白蓮路141號

台11線景點｜成功漁港

號稱世界級戶外地質景觀的石梯坪風景區，只有令人嘆為觀止能夠形容的震撼，壯觀且帶有氣勢，原來這麼美的地景在台灣就能看見，現場還不乏許多外國旅客前來朝聖，身為台灣囡仔一定要來看過一次，也可以順遊附近的項鍊海岸工作室，和夢幻的月牙鞦韆美拍哦！

地址：花蓮縣豐濱鄉石梯坪52號

台11線景點｜月洞

說到台東最美的產業道路，肯定很多人心中會出現伯朗大道的名字，但你知道台11線上也有一條不輸給伯朗大道的道路哦！金剛大道位於台東長濱，比起伯朗大道更安靜，沒有太多的遊客，可以儘情拍照不怕被路人亂入鏡頭。筆直的金剛大道一路沿伸到了海邊，以遼闊的大海為背景，配合兩側的稻田景觀，隨著季節的變換，不論是綠色的秧苗，或是金黃色的稻穗，都是令人驚嘆的鄉村美景。

地址：臺東縣長濱鄉東13鄉道(約台11線85k處)

台11線景點｜烏石鼻

來到了靠近海邊的台11線，當然不能錯過新鮮又便宜的海鮮啦！成功漁港身為台東最大的漁港，又俗又青的海產便是此處的特產，其中又以10月的旗魚季最為出名，港口邊的海產店都有自己拿手的旗魚餐點，吸引各地遊客前往大快朵頤。成功漁港的盡頭有著號稱離海最近的郵筒，漆成深藍色的老式郵筒文青感十足，有機會來到這裡，不妨寫上一張明信片寄給成功與自己吧！

地址：臺東縣成功鎮港邊路19號(約台11線116k處)

台11線景點｜都歷遊客中心

入住民宿看海又不想離市區太遠，OwlStay鹽花是個不錯的選擇，住於花蓮市區同時又緊臨海邊，鹹鹹的空氣隨著海浪聲飄進了純白的房內。如果不說這是間民宿，應該不少人會把鹽花誤認為高雅的咖啡廳吧，白色的牆面、搭配木質傢俬和淺色的裝潢，在暖色調的燈光下，給人一種放鬆安心的感受。

地址：花蓮縣花蓮市海濱街71號

慢點海景民宿

位在太麻里的海道民宿外觀是非常樸實的平房，正面對著蔚藍的太平洋，因為地勢較高，在房門前的空地，就可以居高臨下的欣賞東海岸的美麗大海。早晨起床可以到擁有大片玻璃窗的餐廳，一邊享用老闆娘自製的中式早餐，一邊欣賞陽光照射在大海上波光嶙峋的樣子，以及火車駛過海邊，如同明信片般的美景。

地址：台東縣太麻里鄉11鄰21號

海平面民宿

烏石鼻壯觀的火石融岩好像一顆顆黑色松露落在了美麗的東海岸邊，白色的海浪拍打在礁岩上，激起了飛散的浪花，陣陣的海浪聲令人陶醉，在此觀浪喝杯冷飲最悠閒不過了。由於火山岩的表面有許多坑洞，成為了眾多小魚蝦躱藏的地方，是觀賞潮間帶生物的絕佳地點。烏石鼻同時也是磯釣高手的天堂，尤其是每年二月到了俗稱「白毛」的南方舵魚季節，總能看到不少釣客前來共襄盛舉。

地址：臺東縣長濱鄉(約台11線98k處)

台11線景點｜比西里岸幾米園區 


地址：花蓮縣豐濱鄉磯崎村台11線31.5K

台11線景點｜大石鼻山步道

位於磯崎海水浴場旁的大石鼻山步道，以木棧階、石板、石板階組成，沿途綠樹成蔭可觀察到很多原生植物，就像是大自然的生態寶庫。由於之前步道坍塌，所以只能通行到第二涼亭後原路折返，距離從原本的一公里變成不到四分之一的250公尺，爬到最上面更只要15分鐘，輕鬆就能登頂，不過沿途的美麗風光依然不減，一樣能一覽蔚藍海灣、欣賞太平洋與天空連成一線的豐濱東海岸燦爛海景。

地址：花蓮縣豐濱鄉台11線38.5K

台11線景點｜豐濱新社梯田

海平面民宿位於台東旅遊熱區富岡漁港附近，海景第一排的絕佳位置，讓住在海平面民宿的遊客，不必出門躺在房裡就能欣賞到最棒的台東海景哦！民宿的每個房間都有不同的主題，不論你是喜歡日式風、摩洛哥風、地中海風，這裡通通都有，尤其與大海和白浪超級搭的地中海房最熱門，如果想在此拍出美美的打卡照，記得一定要提早訂房才行哦！

地址：台東縣台東市富岡街52巷23號

前沿拓展：成功漁港,樟原長老教會

【东部海岸】

台湾的东部，我们称为“后山”。

相较于台湾的西部，这里人口稀少、交通不便、资源落后；

但这里却拥有最美丽的风景，倚著海岸山脉，面向著太平洋。

如果你由南向北，穿过迂回的南回公路之后，

迎面而来的是太平洋的风，映入眼帘的是专属于花东的蓝。

沿著台11号公路走，常常惊艳于海洋的美丽，而随时想停驻脚步。

就请你们跟著我的脚步，由南向北，一步步探索这美丽的太平洋海岸！

(我们将这条海岸线分成台东与花莲两个部分来介绍。)

【台东地区】

五、成功镇

# 东部海岸国家风景区管理处 :

位于成功镇都历的东海岸国家风景区管理处，入口处在台11线126公里附近，是交通部管理整个花东海岸旅游景点的大本营，许多台东海岸线的观光休闲设施，都是东管处努力规划的成果，也是旅客获得东海岸丰富旅游资讯的最佳去处与旅游据点。

馆内设置自然地质博物馆、阿美族文化展示区、史前文化、鲸豚馆，游客服务中心有容纳225人的多媒体简报室，每日按时播放东海岸风光，介绍东海岸的各种特色，服务台可购得所有东海岸的自然与人文丛书，提供游客更多的丰富资讯。

东海岸国家风景区管理处背山面海，北望可见成功镇和三仙台，南眺可看见绿岛，四周景观绝佳。

# 成功新港渔港 :

成功新港渔港是东海岸地区最大的渔港，渔港面临太平洋，是黑潮暖流经过的海域，鬼头刀、旗鱼、鲣鱼等丰富的海洋资源，造就了成功独特的镖旗鱼技艺，以及渔业文化的蓬勃发展。当每年10月吹起东北季风时，正是旗鱼的盛产季节，渔民镖旗鱼的独到技艺，让旗鱼肉质保留了鲜Q的口感，深受各地老饕的喜爱。

黑潮暖流经过的成功，也是赏鲸的好地方！搭乘赏鲸船除了欣赏美丽的东海岸岛屿，还有机会欣赏到海豚成群在海中跳跃。除了冬天大啖旗鱼之外，成功镇上还有各种特色小吃、伴手礼，多汁的鬼头刀鱼排、绵密的成功豆花、用当地食材制作的手工饼等等，都是来成功不可错过的美食！

# 成功海滨公园 :

成功滨海公园邻近三仙台，是当地居民散步、骑单车的好地方，也是单车族骑车游东海岸休息的中继站。成功滨海公园串联了三仙台、新港渔港、成功海洋环境教室、台东海洋梦想馆与比西里岸等景点，并结合赏鲸、环境教育、生态及部落深度导览、自行车运动、迎曙光、草地音乐会等多样游憩活动。

# 三仙台 :

位于台东县成功镇东北方的三仙台，是由离岸小岛和珊瑚礁海岸所构成，三仙台岛的地质属于都峦山集块岩，原来是一处岬角，因海水侵蚀逐渐断了岬角颈部，而成了离岸岛。早期这里为阿美族人养羊的地方，因此三仙台又称做Pisiran（比西里岸）。因岛上奇石分布，其中有三块巨大的岩石，传说吕洞宾、李铁拐、何仙姑曾登临此岛，因而得名。

三仙台全岛面积约22公顷，最高点海拔约77米，设有环岛步道，悠闲浏览约需2小时。岛的四周珊瑚礁环绕，由于强烈的风化和海蚀作用，让岛上除了仙剑峡、合欢洞等与三仙台故事有关的天然奇景外，还且散布着海蚀沟、壶穴、海蚀柱、海蚀凹壁等海蚀景观，登岛游览颇富寻幽访胜之趣。自2019年岛上设置太阳宝宝许愿邮筒，让游客登岛许下愿望、实现梦想。

# 比西里岸 :

比西里岸Pisirian”位于成功镇三仙社区一个滨海的小部落，过去为阿美族人养羊的地方，在部落中处处可见几米的画作、由漂流木创作的羊群，以及享誉海外的pawpaw鼓乐团，让这个小小的部落充满童趣。

# 石雨伞 :

石雨伞是东海岸十分有名的地质景观，位在成功镇北方10公里的海岸，长达1公里的海岬，属于石灰岩隆起的海蚀地形，有着嶙峋的礁石、狭长岩岬、平衡岩、壶穴、海蚀门等丰富的海蚀地貌。其中以平衡岩最为著名，一块比较细长的岩石稳稳托住了较大的珊瑚礁岩，形成一种特殊的平衡结构，看起来就像一把雨伞，因而称为石雨伞。

从凉亭眺望石雨伞，顺着一旁的木栈步道走，来到海水清澈的小船澳，便是当地人戏水的天堂！有着石雨伞天然的屏障挡住外海的海浪，让小船澳成为安全的戏水地点。

# 堺桥休息区 :

位在台东县长滨乡台11线约99公里处，是长滨乡与成功镇的交界地带，设有一座面海凉亭，常有游客来到这里观海景吹海风、欣赏日出日落，还能眺望远处的三仙台。

六、长滨乡

#乌石鼻 :

乌石鼻位于长滨乡南方宁埔村一片伸入海中的岬角，因远看像人的鼻子向外突出而命名。迷人的乌石鼻拥有为全台湾面积最大的柱状火山成岩体，突出于海中的黝黑岩体，便是火山活动熔岩喷发的产物。乌石鼻四周有着多样化的潮间带生态与海蚀地形，加上海域附近有丰富的浮游生物和鱼，更是成为东海岸矶钓和潜水的天堂。

# 金刚大道 :

长光部落位于台东县长滨乡，是个背山面海的美丽部落。进入长光部落，沿着东13线长光产业道路一路往西直行，笔直的“金刚大道”旋即映入眼帘。站在大道上，南北两侧尽是无边际的梯田，层次分明地坐落于金刚山脚下与太平洋畔。海风吹起，金黄色的稻穗随风摇曳，掀起片片稻浪，和海涛声相互呼应，无边际的壮阔视野，可与纵谷的“伯朗大道”相媲美。

部落里最负盛名的，是位于西侧海岸山脉的“金刚山”；在山势地形与天光云影的巧妙作用下，仿佛一只巨大的大猩猩矗立在山头，“金刚山”因而得名，也成为长光社区的重要地标。

# 八仙洞 :

八仙洞以兼具地质景观与史前遗址闻名，拥有独特的海蚀洞景观，也是迄今台湾最古老的史前文化~~长滨文化的遗址之一，属于旧石器时代先陶文化，距今约三万年前左右，被指定为国定考古遗址，是台东县位置最北端的大型游憩景点。

八仙洞自然形成的十数个海蚀洞穴，因八仙过海传说而得名。这些洞穴原本在海面上，经过东海岸地壳陆升作用，加上海浪冲蚀，岩壁较松软的部分被侵蚀形成洞穴。各个海蚀洞穴均设有环状木栈步道可以通往，其中最大的洞穴是灵岩洞，属于直立型海蚀洞穴，目前仍在继续增高中。离地面最高的是昆仑洞，高度有130米；其他还有乾元洞、朝宸洞、海雷洞、潮音洞、永安洞、水帘洞等。

#旧樟原桥游憩区 :

樟原是台东县海岸最北端的一个村落，是日本殖民统治台湾期间是樟脑的重要产地。为开发东部的樟脑资源，于1937年在横跨姑仔律溪上兴建第一座樟原桥，有三个拱形桥墩，展现古朴天成的自然风韵。后来配合东海岸道路拓宽，于1986年在樟原桥上方另建新桥，1998年时再配合东部海岸公路(台11线)公路截弯取直兴建第三座桥梁，三桥并立于姑仔律溪，见证了东海岸公路发展史，也表现了三座不同时期的美学观点与工程技术，旧樟原桥所拥有的珍贵价值已被列历史建筑的文化资产。

作者简介：台湾导游小鱼儿，本名叶瑾瑜，台湾知名导游，台湾观光旅游产业协会副理事长，胞兄叶匡时曾担任台湾“交通部”部长，高雄市副市长等职务。
