澎湖首屆國際燈光藝術節明(7)日點亮「燈」場

今年澎湖國際燈光藝術節主燈為「舞躍銀波」，由五隻海豚組成，活動期間每週五至週日晚間7時到9時，每半小時將演出璀璨主燈秀，隨著燈光360度旋轉並上下跳躍，可愛活潑的樣子象徵澎湖的律動和生命力，另外活動現場也有光耀漁翁島、洄游豐收等大型主題燈座和互動遊戲裝置，並規劃繽紛亮麗的光環境燈區，後續也將陸續於十月下旬和十一月下旬推出新的主題燈座，歡迎民眾在不同時節回到國際廣場賞燈。

活動期間每週五到日，晚間7時至9時也將安排街頭藝人展出，而明(7)日開幕當晚，主辦單位規劃「歌頌美麗漁翁島」音樂秀、另外晚間7時起縣內知名街頭藝人陳冠鳴老師和「遊手好絃樂團」、「仙人掌輪椅舞團」、「盲人街頭鼓手」等表演團體將在場地內演出，晚間8時將演出首場璀璨主燈秀。

【旅遊經 編輯部報導】

最後提醒前來賞燈的民眾，除了可於國際廣場停車場、第三漁市場對面停車場及文化局停車場停放車輛外，尚有需收費的南海遊客中心停車場、海埔路停車場以及馬公市水源路停車場可多加利用，也提醒活動期間禁止車輛進入國際廣場。提醒，因天氣預報開幕當日風速強勁，原訂開幕當天演出的無人機秀將取消表演，但璀璨主燈秀還是照常舉行，另外開幕當日晚間7時將在服務台（入口涼亭旁）臉書按讚贈送黑糖糕。

除了燈光藝術及街頭藝人演出，主辦單位規劃6場達人手作體驗課程及6場藝文講座等系列活動，全程免費，陸續開放報名中，相關訊息可至澎湖Travel網站(http://penghutravel.com/)，或澎湖國際燈光藝術節臉書粉絲專頁查詢！

在活動登場前，透露欣賞澎湖國際燈光節的小秘訣，國際廣場的天橋是視野絕佳的地點，可一覽活動全區及港灣漁火美景，沿著新生路人行道樹的「漂浮光廊」有滿滿的光球，觸手可及的光點彷彿即將降落澎湖，靠近國際廣場停車場的「海島星空」打造一片超大型燈海，鄰近澎湖福朋喜來登酒店的「蒲公英花丘」不管從哪個角度拍，都能被五彩繽紛的燈光包圍。

澎湖首屆以燈光為主題的「澎湖國際燈光藝術節」於明(7)日開幕，活動中首次利用國際廣場空間作為活動主辦地點，展期長達近5個月，將持續至明(2023)年2月28日止，分為三期燈區展出，每一期在原有燈區以外還會加上新的燈區佈置，近期主辦單位正在國際廣場內緊鑼密鼓施工，吸引許多民眾夜間前往駐足賞燈，更創造不少拍照亮點。

前沿拓展：DEFAULT

在TCP/IP网络中，默认网关又叫缺省网关或缺省路由器，英文名为DefaultGateway，是子网与外网连接的设备，通常是一个路由器，而该路由器的IP地址即为网络配置参数的默认网关地址。通常地，IP地址192.168.1.X的默认网关是192.168.1.1。

下面我将利用手机和电脑分别演示如何查看与设置网络配置参数中的默认网关。

演示机型如下：

电脑：联想天逸510s、Windows10专业版

手机：小米12、MIUI13.0.21稳定版

电脑端查看、设置默认网关的方法查看方法1

在Windows任意界面内，按下键盘win+x组合键，在弹出的菜单中鼠标单击网络连接选项，打开网络与Internet设置界面，点击页面中的查看硬件与连接属性，在弹出的属性页面中即可查看当前网络参数中的默认网关。

查看方法2

在Windows桌面下，鼠标点击开始菜单中的设置选项，打开Windows设置主界面，点击网络和Internet菜单选项，或在右键单击Windows状态栏右下角网络图标，选择打开“网络和Internet”设置，在弹出的网络状态查看页面内点击属性按钮，进入到以太网属性页面，就可以看到当前网络参数中的默认网关。

查看方法3

在Windows桌面下，鼠标右键单击桌面网络图标，然后选择属性，在弹出的网络和共享中心设置界面内选择更改适配器设置，打开网络连接查看页面，鼠标双击以太网连接图标或右键单击后选择状态，在弹出的以太网状态页面内，即可查看当前网络参数中的默认网关。

查看方法4

在Windows下，按下键盘win+R组合键，在弹出的运行输入框中输入cmd，然后回车，或鼠标依次点击开始菜单>Windows系统>命令提示符，打开命令提示符程序，在命令提示符下输入：ipconfig /all (请注意：反斜杠“/”前面有一个空格），按下回车后，在命令提示符程序列出的Windows IP配置列表中，也可查看默认网关。

查看方法5

因为默认网关地址就是网络中路由器的IP地址，而路由器的IP地址又是路由器的默认管理地址，所以，在用户未修改路由器默认管理地址的情况下，路由器的默认管理地址即为网关地址。换句话说，默认网关、路由器IP地址和路由器管理地址，这三个是相同的。一般地，路由器的默认管理地址在其背面标签上有明确标注，通常为192.168.1.1或192.168.0.1。

修改方法1

鼠标右键点击Windows桌面状态栏右下角网络图标，选择打开“网络和Internet”设置选项，

修改方法2

在Windows开始菜单中点击设置按钮，然后选择网络和Internet选项，在弹出页面内找到高级网络设置，点击更改适配器选项，在打开的网络连接列表中找到以太网图标，右键选择属性，在弹出的对话框中，双击列表中的Internet协议版本4（TCP/IPv4），在弹出的属性对话框中，可以选择自动获取IP地址或者固定IP地址，在选择使用下面的IP地址选项时，就可以修改默认网关地址。

特别提示：电脑端修改默认网关，是在明确知道网络默认网关的情况下，才可以修改，否则，随意地修改电脑中默认网关，会导致电脑无法正常上网。如果我们修改网关后，电脑提示无法连接至Internet，只需要在上述修改方法中打开的设置界面中选择自动获取IP地址，系统将自动检测并设置正确的网关。

手机端查看、设置默认网关的方法查看方法

当手机已经连接至路由器的WiFi后，在手机设置里面即可查看该WiFi网络的默认网关，同时也是连接至该路由器下所有的设备网关地址。

打开手机设置，找到WLAN设置界面，或在手机屏幕最上方下拉调出控制中心，长按无线开关按钮，进入到WiFi列表中，点击当前连接的WiFi名称后面的小箭头，打开WiFi网络详情，页面中显示的路由器一项，即为当前网络的网关地址。

修改方法

手机修改网关的方法比较简单，在上述网关的查看界面内，滑动页面并找到IP设置选项，点击该选项选择静态，即可出现修改IP地址的表单选项，表单中路由器选项即为网关，在该输入框中即可修改网关地址。

以上就是查看和修改网络中网关的具体方法，这里需要注意的是：如果路由器的IP地址被修改后，默认网关随即被改变，该路由器下以固定IP地址或静态IP地址联网的电脑和手机用户将无法上网，这些用户需要修改正确的网关地址才可恢复上网。所以，为避免用户频繁地修改每一台设备，建议用户将手机和电脑设置为自动获取IP地址。

拓展知识：DEFAULT

尽管 Ampere 的 Altra Max CPU 已经推出有段时间，但知名超频专家Der8auer 还是决定拿一枚80核 ARM 处理器进行测试，以评估它在标准 PC基准测试中的表现。可知 Altra Max Q8030拥有80个 Neoverse-N1内核，并在Geekbench 跑分项目中超越了 AMD 高端线程撕裂者（Threadripper）SKU 。

据悉，Dera8uer 是在德国大型数据服务提供商 Hetzner的协助下拍摄了这支视频，该公司使用 Ampere 的 Altra Max CPU 为其服务器提供支撑。

Altra Max Q8030的时钟频率为3.0GHz，最大 TDP功耗为210W、具有128条 PCIe4.0+ 八通道 DDR4-3200内存系统，此外 Ampere 还有一款主频高达3.3GHz的 Q8033

测试选用了 LGA4926插槽的参考设计主板，较 AMD 线程撕裂者的 TRX40/ WRX80平台的LGA4096插槽要大得多。

服务器采用了主动式散热风道散，辅以巨大的工业级铜热管+铝块散热器。

主板两侧为8通道 /16条 DDR4DIMM 插槽，最高支持4TB 容量。

性能方面，Linux 平台上运行的 Geekbench5基准测试显示，Altra MaxQ8030的空闲功耗在30-40W，加上工作负载后会升至95-100W、且平均温度约50℃ 。

不过需要指出的是，这不仅仅是 CPU 本体的功耗参数。在运行 Geekbench5时，Altra MaxQ8030消耗了100-105W 左右的功耗，温度在50-55℃ 。

在80个内核的多线程测试中，额定峰值功耗达到了310W，温度也上升到了70℃ 。

最终 Ampere Altra Max Q8030在Geekbench5基准测试中的单核得分为882、多核成绩为44425。

作为对比，拥有64C /128T 的 AMD Zen2锐龙线程撕裂者3990X 的多线程性能为34735分，意味着 AltraMax Q8030具有至少28% 的领先优势。

此外 ARM CPU 的能效表现也相当喜人，CPU 本体仅消耗170-180W 功率，约为线程撕裂者的1/2。

与此同时，TechTechPotato 在评论中添加了 AMD Ryzen Threadripper5995WX的性能数据（来源:@FlanK3rXS）。

（截图 via WCCFTech）

最后，开盖表明 Ampere Altra Max Q8030CPU 采用了基于 TIM的焊料、且芯片尺寸达到了573.75m㎡（22.5×25.5mm）。

这较核心更少的英特尔旧 Skylake-X 部件（480m㎡）大了一圈，但在 ARM 服务器 CPU生态开始崛起的同时，x86领域仍由 AMD 和 Intel 所主导。
