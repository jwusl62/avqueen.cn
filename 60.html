瀑布、溫泉、湖泊、絕壁??與不同自然元素相輔相成的日本絕景紅葉聖地！

高千穗峽（たかちほきょう）是高千穗的象徵，這座大約在10萬年前因阿蘇火山噴發，大量火山碎屑沿五瀨川往下流，急速冷卻後形成的峽谷，柱狀節理的火成岩歷經河水的長期侵蝕，最終被刻畫出一片高80~100公尺、連綿長達7的斷崖。


高千穗峽（Image Source : Getty Creative/iStockphoto）

高低落差17公尺的真名井瀑布，出現在高千穗峽一處非常狹窄的地點，由上方公園水池的閘門控制水源。划船至此，可以看見彷彿一線天般的特殊景致，並且能近距離仰望瀑布，感受飛濺臉上的水珠！

以襯著藍天的富士山為背景，湖畔群山層疊環繞，再加上朱紅色的箱根神社鳥居畫龍點睛，蘆之湖的景觀不只充滿色彩也展現層次，如此迷人的風景如果只是從旁觀看，未免可惜，因此湖上穿梭著多種觀光船，讓遊客能「深入其境」??


蘆之湖（Image Source : Getty Creative/iStockphoto）

＊九州?高千穗峽 （11月下旬～12月上旬）


高千穗峽（Image Source : Getty Creative）

另一條路面步道串連起二湖到五湖，這條步道開放時間視月份不同。此外因為鄰近棕熊棲息地，在棕熊活動期和植生保護期，必須在專業導遊的帶領下才能前往，同時有人數限制。不過所幸賞楓期間不在這些限制時段之中，大家可以自由在群山與湖泊之間，漫步於樹葉變色的原始森林中～～

蘆之湖（あしのこ）大約形成於3,000年前，當時火山大量噴發造成的山崩，堵住了早川的去路，於是誕生了這座湖泊。表面積約7平方公里，它是神奈川縣的第一大湖，水源主要來自於湖底的湧水。至於蘆之湖為什麼名氣響亮，因為它同時也是遙望日本境內最高峰——富士山的名所～～


蘆之湖（Photo by Dukas/Universal Images Group, Image Source : GettyEditorial）

身為箱根的象徵，蘆之湖在秋天也沒讓人失望，滿山遍野的楓葉刷出金黃、橘紅、豔紅、棕紅等漸層色調，讓它顯得更加如詩如畫。如果有機會來到這裡，千萬別錯過拍張與楓紅、富士山、鳥居大合照的機會！

占地38,633公頃，這座國家公園陸域面積一半以上屬於「特別保護地區」，除原始景觀外，還能看見棕熊、北海道狐狸、海豹、海雕等野生動物，當中最知名的景點是知床五湖（しれとこごこ）。


知床五湖（Image Source : Getty Creative/iStockphoto）

桂川流經京都時，在西邊切出了右岸的嵐山和左岸的嵯峨野，不過近年來這兩個區域已經不再那麼壁壘分明，大多以渡月橋為中心，將其左右兩岸周邊地區並稱為嵐山??


嵐山（Image Source : Getty Creative/iStockphoto）

橫跨神奈川縣和靜岡縣的箱根山，是一座至今依舊有火山活動的活火山，也因此山麓和山腰有許多溫泉，箱根就是其中非常知名的一座。這座溫泉鄉還有另一大亮點，正是昔日破火山口形成的堰塞湖。


蘆之湖（Image Source : Getty Creative/iStockphoto）

嵐山不能錯過的還有嵯峨野小火車（嵯峨野トロッコ列車），這條全長7.3公里的觀光路線，採用舊貨車或客車車廂改裝成的開放式列車，窗戶沒有玻璃，讓遊客可以更接近沿途景物。駛進普通列車無法前往的深山溪谷，賞楓之餘，還能呼吸新鮮空氣同時感受徐徐微風。

因為地點隱密，過去這裡是當地居民才知道的溫泉療養場，不過1980年代吹起一陣秘湯熱，這座保留「日本原始風貌」的溫泉鄉突然聲名大噪，如今成為深受國內外遊客喜愛的泡湯勝地！


乳頭溫泉鄉（Image Source : Getty Creative/iStockphoto）

創建於1461年的寶嚴院（ほうごんいん），歷經戰亂燒毀、土地被收購等問題，最後在現在的位置重建。院內的「獅子吼之庭」是一座借景回遊式庭園，有塊獅子狀的岩石，並且以突破岩石生長的松樹著稱。這座庭園是欣賞夜楓的好去處，從17:30開始點燈，大約開放約3小時，遊客可以在涼爽的秋夜，觀看燈光與紅葉大玩影子遊戲～～


嵐山－嵯峨野小火車（Image Source : Getty Creative/iStockphoto）

＊關東?蘆之湖 （10月下旬～11月中旬）


蘆之湖（Image Source : Getty Creative/iStockphoto）

隱藏於山腳，位於秋田縣東部仙北市田澤湖地區的乳頭溫泉鄉（にゅうとうおんせんきょう），是多處錯落十和田八幡平國立公園乳頭山山麓溫泉的總稱，四周環繞著原始且茂密的日本山毛櫸森林，感覺就像一處秘境。


乳頭溫泉鄉（Image Source : Getty Creative/iStockphoto）

＊北海道?知床五湖 （10月中旬～11月）


知床五湖（Image Source : Getty Creative/iStockphoto）

1934年時成為日本國家級名勝景點，宏偉的山壁給人充滿魄力的感覺，峽谷內有條沿五瀨川延伸、約1公里長的步道，沿途經過五瀨川最窄處的「槍飛橋」、3公尺高的「鬼八力石」等景點，秋天時漫步於紅葉之間，景色非常迷人～～


高千穗峽（Image Source : Getty Creative）

古剎也是嵐山的一大特色，造型猶如五重塔的常寂光寺（じょうじゃっこうじ），種植了將近200棵楓樹，從茅草屋頂的仁王門，到通往本堂的石階參道，每到秋天全都染成一片赭紅。落成於1339年的天龍寺（てんりゅうじ）不只是世界遺產，寺內的曹源池庭園，池水倒映了五顏六色的紅葉，更是嵐山最佳賞楓角落之一。


嵐山－寶嚴院（Image Source : Getty Creative/500px）

橫跨桂川的渡月橋，如今以鋼筋混凝土打造，然而外觀採用復古的木造護欄，木橋般的造型與景色融為一體。你可以搭乘屋形船，讓船夫帶領你進入五彩繽紛的水上世界，河流兩旁的山林彷彿被抹上斑斕的顏料，你會以為自己走進了秋天的童話～～


嵐山－天龍寺的曹源池庭園（Image Source : Getty Creative）

＊關西?嵐山 （11月下旬～12月上旬）


嵐山（Image Source : Getty Creative/EyeEm）

大部分位於栃木縣的日光國立公園（にっこうこくりつこうえん），坐落那須火山帶所在的山嶽地帶，當中日本百名山嶽之一的男體山，是一座圓錐狀的成層火山，從它噴發形成的熔岩上方，有座瀑布傾瀉而下，龍頭瀑布（りゅうずのたき）是日光最具有代表性的瀑布之一。


龍頭瀑布（Image Source : Getty Creative）

屬於溪流瀑布的它綿延了210公尺，先是以平緩的姿態流向下游，沿途經過寬約10公尺的梯狀岩層後，湍急的水流在接近深潭處，又被一塊巨大的石頭一分為二，於是瞬間從頂端一鼓作氣奔流而下，產生令人驚艷的非凡氣勢！


龍頭瀑布（Image Source : Getty Creative）

不過「使用」高千穗峽的最佳方式，是租艘遊覽小船在峽谷間泛舟。隨水流緩緩漂移的同時，邊抬頭欣賞點綴於斷壁間的美妙秋色，在獲選為日本百瀑之一的真名井瀑布（まないのたき）附近，就能看到絕美的楓紅。


高千穗峽（Image Source : Getty Creative）

日本政府目前入境防疫觀光政策，除10月11日取消入境人數限制外，同時開放海外旅客自由行和短期免簽證旅客入境，被列為低風險國家的台灣，國人可以以三劑疫苗証明，或72小時內PCR陰性證明，免除日本入境檢疫隔離。之前我們曾經介紹過日本6大秋紅名勝景點，除了前往名園名院、寺廟城堡等人為景觀中欣賞楓葉之外，日本的紅葉狩，當然也不能錯過與自然美景相互結合的賞楓聖地！氣候越冷的地方，樹葉越先變色，因此紅葉由北往南、從山上朝山下一路擴散。只不過日本賞楓地點不勝枚舉，你是否出現選擇障礙？沒關係，來看看小編的Pick：你只需要煩惱想邊泡湯、還是以磅礴的瀑布為背景、將富士山和鳥居一同盡收眼底、或划船穿越宏偉的斷崖，邊欣賞楓葉？

乳頭溫泉鄉總共有10處以上的泉湧，當地搭蓋茅草屋頂、洋溢古早日式風情的7間溫泉旅館，各有各的泉源，也因此光是待在這裡，就能泡到多種泉質的溫泉：「鶴之湯溫泉」是乳白色的溫泉、「乳頭溫泉鄉妙乃湯」則是對肌膚具有療效的硫酸鹽泉??


乳頭溫泉鄉（Image Source : Getty Creative/iStockphoto）

秋天本來就是適合泡湯的季節，更棒的是，這裡的溫泉旅館都有露天溫泉，也就是說你可以一邊泡湯，一邊欣賞四周的紅葉，伴隨著古老的日式氛圍，以及山林幽靜的空間，感受遠離塵囂、甚至時空的「逃離」～～

而它之所以稱為龍頭瀑布，據說是因為那塊「從中作梗」的石頭形狀類似龍頭，另一種說法是，分成兩道的瀑布彷彿龍的兩條鬍鬚??這座瀑布與中禪寺附近的華嚴瀑布，以及湯之湖南端的湯瀑布並稱「奧日光三大瀑布」，是當地的人氣景點。


龍頭瀑布（Image Source : Getty Creative）

＊東北?乳頭溫泉鄉 （10月下旬～11月中旬）


乳頭溫泉鄉（Image Source : Getty Creative/iStockphoto）

＊關東?龍頭瀑布 （10月上旬～10月下旬）


龍頭瀑布（Image Source : Getty Creative/iStockphoto）

北海道東北部的知床半島突出於鄂霍次克海，它在1964年成為國立公園，並且在2005年時因「北半球能看見浮冰的最南地區之一」，被登錄為世界自然遺產。名稱來自於愛奴語的它，有著「地球突出之處」、「地球的盡頭」（シレトク）的意思??


知床五湖（Image Source : Getty Creative/iStockphoto）

根據日本傳說，地位最高的天神——天照大神，指派自己孫子天津彥彥火瓊瓊杵尊（日本第一代天皇神武天皇的曾祖父）下凡統治人間，而祂降臨的地方正是今日宮崎縣北端的高千穗！也因此這裏成為日本創生神話的發源地，至今仍有許多傳說都以它為背景??


高千穗峽（Image Source : Getty Creative/iStockphoto）

打從平安時代以來，這裡就是許多貴族別院的所在地，經常成為古典文學作品和歷史故事的舞台，再加上桂川兩旁種滿櫻花與楓樹，讓它成為一年四季都展現不同風情的熱門景點。秋天來到這裡賞楓，不妨以桂川和渡月橋為起點。


嵐山（Image Source : Getty Creative）

5~6月時，龍頭瀑布四周開滿紫紅色的杜鵑，景色已相當迷人。然而到了秋天，9月下旬開始進入紅葉季，環繞瀑布的森林可以欣賞到紅艷艷的楓葉和金黃色的華東椴，繽紛的色彩讓人彷彿置身畫中。此外，從觀瀑台還能遠眺被紅葉環抱的瀑布潭，絕美的景致難以形容～～

知床五湖是知床八景之一，總共由五座分別編號1~5號的湖泊組成，錯落於一片茂密的森林中，以知床山脈融雪而成的地下水為水源。有兩條步道分別前往這五座湖泊，環繞一湖的高架步道全長800公尺，擁有多座可以分別欣賞知床連峰、鄂霍次克海、湖畔的展望台，以不同角度飽覽湖光山色。


知床五湖（Image Source : Getty Creative/iStockphoto）
