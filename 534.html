「濱海振興」周末小旅行於10/22~11/6的周六、日辦理，由桃園將補助500元/人，一日遊行程只需自費999元，行程包含橫山書法藝術館、國巨洋傘文創園區、竹圍漁港美味海鮮、陽榮休閒農場或遊艇體驗。其中10/22、11/06遊程可登上造價上千萬的西莉亞號遊艇體驗。

大園竹圍漁港一直以來以新鮮魚貨著名，近幾年桃園市政府持續推動竹圍漁港各項建設，讓傳統的漁港逐步轉型為具有國際遊艇停泊與觀光旅遊之熱門景點，其中如漁港北邊入口的「彩虹橋」是熱門拍照點；新建的「竹圍漁港薄膜天幕廣場」，除可提供遊客遊憩空間，常舉辦多項活動，提高觀光休閒功能；旁邊的「遊艇碼頭」除執行漁船和遊艇的修船業務外，亦可容納120艘遊艇。桃園市政府為提振濱海旅遊觀光，於明(22)日起至11月6日為期三周舉辦「2022樂遊竹圍漁港」桃園濱海系列活動！

竹圍漁港

竹圍漁港

「2022樂遊竹圍漁港」活動以「濱海振興」與「海洋教育」兩個主題展開，首先明(22)日於竹圍漁港天幕球場，由知名型男大主廚吳秉承現場料理竹圍新鮮海鮮揭開系列活動，還有許厝港溼地生態導覽、豬鼻子沙灘淨灘活動、減塑市集、SDGs海洋教育闖關遊戲、以及超值振興旅遊行程。

「海洋教育」系列活動包含減塑市集、SDGs闖關活動以及精彩藝人表演、10/29豬鼻子沙灘淨灘、10/23、10/29、10/30許厝港生態導覽。豬鼻子沙灘百人淨灘，希望在振興觀光的同時也能換起民眾對海洋教育的重視。參與活動的民眾有機會獲得在地農特產品，還有機會抽中iPhone14及各項豐富好禮獎。

相關活動訊息，請至桃園觀光導覽網(http://travel.tycg.gov.tw/)查詢。

大家好，我是阿常，上一篇阿常讲了《数据库 SQL 约束之CHECK》，今天阿常和大家分享《数据库SQL 约束之 DEFAULT》。

SQL约束的定义

SQL约束用于规定表中的数据规则。如果存在违反约束的数据行为，行为会被约束终止。

约束可以在创建表时规定（通过 CREATE TABLE语句），或者在表创建之后规定（通过 ALTER TABLE语句）。

SQL DEFAULT 约束

DEFAULT约束用于向列中插入默认值。

如果没有规定其他的值，那么会将默认值添加到所有的新记录。

一）CREATE TABLE 时的 SQL DEFAULT约束

数据库实例

CREATE TABLE student

(

IID int NOT NULL,

name varchar(255) NOTNULL,

sex varchar(255),

age varchar(255),

country varchar(255) DEFAULT\’CN\’

)

通过使用类似 GETDATE() 这样的函数，DEFAULT约束也可以用于插入系统值：

数据库实例

CREATE TABLE orders

(

OrderIdvarchar(255),

OrderNovarchar(255),

P_Id varchar(255),

OrderDate date DEFAULTGETDATE()

)

二）ALTER TABLE 时的 SQL DEFAULT 约束

数据库实例

ALTER TABLE student

ADD country SET DEAFAULT\’CN\’;

三）撤销 DEFAULT 约束

数据库实例

ALTER TABLE student

ALTER country DROPDEFAULT;
