前沿拓展：駐站達人,快樂雲

（报告出品方/作者：东北证券，邓利军、杨正旺）

1. 虚拟人是什么？

1.1. 虚拟人定义—多重人格特征的综合产物

虚拟人是技术手段实现的具备多重人类特征的综合产物。虚拟人指存在于非物理世界中，由计算机图形学、图形渲染、动作捕捉、深度学习、语音合成等计算机手段创造及使用，并具有多重人类特征（外貌特征、人类表演能力、人类交互能力等）的综合产物。市面上也多将其称为为虚拟形象、虚拟人、数字人等。

1.2. 虚拟人分类—文本驱动、AI 驱动、虚拟 IP驱动和虚拟分身

按照虚拟人下游应用场景，虚拟人可分为服务型虚拟人和身份型虚拟人。按照运营方式，又可进一步细分为文本驱动型、AI 驱动型、虚拟 IP 和虚拟分身。

1.2.1. 服务型虚拟人—文本驱动和 AI 驱动

服务型虚拟人强调“替代”真人，按照交互程度可以将服务型虚拟人的主要功能分为 文本驱动型虚拟人和AI 驱动型虚拟人。1）文本驱动型虚拟人旨在减少标准化内容 的制作成本或提高真人曝光度。文本驱动型虚拟人只需输入文字即可完成视频制作。虚拟人和用户之间没有互动或互动较少，更多的是由虚拟人向用户单方面进行信息传输。目前该功能主要应用于新闻播报、产品说明、政务客服等说明性内容和导航导览、展览介绍等简单交互性内容。未来提高问答的转化率是服务型虚拟人该功能的主 要发展方向。

2）AI 驱动型虚拟人主要针对细分场景提供顾问、关怀、陪伴、事务处理等服务。用户通过文字或语音与虚拟助手进行实时对话，虚拟助手会在一问一答中为用户提供专业的咨询服务，该方案可深度赋能文博、医疗、政务服务等领域。随着 XR 或全息投影的发展，具有具体形象的 Al助手将具有较大应用潜力。同时，AI 智 能语音技术伴随着人工智能、智能手机等高成长行业的发展，技术迭代迅速，技术成熟后或对服务型虚拟人的发展带来较大推动。

1.2.2. 身份型虚拟人—虚拟IP 和虚拟分身

身份型虚拟人强调“创造”虚拟身份。从创造目的角度可以将身份型虚拟人分别虚拟IP和和虚拟分身。1）虚拟 IP 可以满足 IP 运营、打造、立体化的需求。虚拟 IP在现实世界中并不存在对应对应真人，其外貌特征、基本人设、各类偏好、背景信息等均 由人为设定，具体包括原创IP和基于文本、漫画、游戏、企业创造的 IP，虚拟歌姬、 虚拟 UP主、虚拟主播、超写实 KOL、品牌虚拟 IP等均属于这一范畴。2）虚拟分身 是为真实的个体创造“第二分身”从而参与虚拟世界。不同于 QQ秀等平面形象，虚拟形象可以反映用户的外貌、表情、语言、神态等多种要素，比平面形象更加完整、 立体，并且给用户带来的参与感更强。在Metaverse 概念下，虚拟人将作为用户在虚 拟世界的虚拟分身参与虚拟活动和社交。目前部分厂商正在搭建能够为用户提供个人虚拟形象的平台，虚拟形象可代表用户参加到 VR 游戏、VR Chat、内容工具等， 典型代表是 Meta 的虚拟会议平台Workrooms。

1.3. 虚拟人发展历程—技术驱动为主，虚拟IP最早落地

虚拟人的发展史就是技术的发展史。虚拟人的发展历史与制作技术的进步高度相关。 CG 技术和 AI的发展贯穿了虚拟人至今的发展历程。1）20 世纪 80 年代，手绘萌芽阶段。该时期内虚拟人诞生，最早在日本以虚拟歌姬的形式诞生，制作形式以 2D、 3D动画为主。同期，虚拟人开始涉足电影行业，影视虚拟人由化妆演员和手绘实现； 2）21 世纪初，CG 探索阶段。该时期内CG和动作捕捉技术兴起，虚拟人开始加速 应用到影视行业，国外的初音未来、《指环王》里的咕噜等是这一时期的代表产物。但虚拟人表现形式仍较粗糙，且以 2B 端影视应用为主，因此应用没有得到更广泛 的推广；3）2016-2019 年，AI初级阶段。深度学习算法的进步带动了虚拟人行业的 发展，各种应用开始落地，服务型虚拟人的应用发展迅速；4）2020 年以来，技术全面提升。随着元宇宙概念的升温，资本加速进入虚拟人市场，促进了技术的快速发展和应用场景的加速探索，服务型虚拟人和身份型虚拟人的应用全面落地。整体来看，虚拟人的发展主要由技术驱动，可通过关注虚拟人产业链上游底层技术的发 展方向来判断虚拟人行业的发展方向。

虚拟 IP 是虚拟人最早落地的表现形态。虚拟 IP 是虚拟人落地较早的应用，经历了虚拟歌姬、虚拟 UP 主、虚拟主播、原创虚拟 IP 四个发展阶段。具体来看：1）2007 年，虚拟歌姬诞生，典型代表是日本的初音未来。虚拟歌姬是音源库搭配虚拟形象的产物， 本质上是由音源库组成的虚拟形象。2）2016 年末，虚拟UP 主出现，虚拟 UP 主起 源于日本的 Vtuber，以日本虚拟 IP 绊爱的出现为标志，目前活跃于日本和中国。3） 2017年，虚拟主播进入中国，虚拟人进入中国市场，和快速发展的电商直播行业结合，催化了虚拟主播的发展。虚拟主播由真人驱动，由真人而不是音源库中的声音文本来做配音，再结合面部捕捉等技术实现。基本形式以主页直播为主，以短视频等其 他形式为辅，互动性和真实性较虚拟歌姬大大加强。4）2020年以来，原创虚拟 IP 崛 起，以 STAR Labs 设计的虚拟 IP NEON 为标志。在技术的持续革新下，虚拟人制作成本不断下降，虚拟歌姬、虚拟 UP 主、虚拟主播、原创虚拟 IP 等协同发展，发展 速度进一步加快。

1.4. 虚拟人商业模式—2B 端快速发展，2C端模式丰富

可根据虚拟人类型对虚拟人商业模式进行划分。服务型虚拟人以 2B 端为主，商业模式以平台为主。身份型虚拟人以 2C端为主，商业模式丰富。目前虚拟人商业模式基本已被验证且用户增速、营收增速持续高涨，有望进入商业模式快速兑现阶段。

1.4.1. 服务型虚拟人商业模式—平台模式为主

服务型虚拟人商业模式以平台模式为主。服务型虚拟人的基本商业模式为虚拟人制作厂商为商用户打造虚拟人，收取定制费用或租赁费。早期服务型虚拟人定制化程度较高，目前随着技术成本的下降和标准化技术的确立，服务型虚拟人制作已向平台模 式方向发展，即可在模板虚拟人的基础上适当定制。2021年以来，科大讯飞、华为、 OPPO、英伟达、蓝色光标等公司纷纷推出了虚拟人生成/交互平台，该类平台使开发者/用户可以通过简化的模式复用相关技术，以较低的成本创造出虚拟人形象。虚拟人平台的推出可以大幅降低虚拟人的制作成本，推动虚拟人的消费级应用落地。未来 服务型虚拟人有望全面替代移动互联网中的 APP、UI界面功能，承载人机交互的职 能。

1.4.2. 身份型虚拟人商业模式—歌姬、UP主、电商、代言、衍生品模式

身份型虚拟人商业化模式丰富。身份型虚拟人中，虚拟 IP 是目前商业化竞争最为激烈的商业模式，其基本模式是 IP 运营公司携手技术型虚拟人制造公司打造虚拟 IP 并进行运营，完成流量积累后在文娱、消费领域变现。随着虚拟人制作技术的发展，虚拟人与热门商业模式的结合愈加紧密，具体商业模式包括歌姬模式、UP 主模式、电商模式、代言模式和衍生品模式，这些商业基本均已得到验证。虚拟分身对应的虚拟 社交平台模式方兴未艾，商业模式尚未成型。

1.4.2.1. 歌姬模式—中之人+语音合成引擎

虚拟歌姬是由人工创造的语音合成引擎为基础开发贩售的虚拟歌手。虚拟歌姬于 2007年诞生于日本，典型代表是日本的初音未来和我国的洛天依。最基本的商业模式是通过发行新单曲和举办演唱会变现。随着人气的积累，虚拟歌姬还可尝试其他商 业模式，如：电商模式、代言模式、衍生品模式等。2017年，“出道”5 年的洛天依在上海举办第一场线下演唱会，演唱会首批 500 张 SVIP 席票价 1280 元的内场票在 3分钟内售罄，超过 100 万人在 A 站观看直播；2020 年洛天依首次直播带货，直播 在线观看人数一度高达 270 万；截至2022 年 1 月，洛天依微博粉丝高达 513.5 万， B 站洛天依官方账号则拥有粉丝 261.8 万，投稿视频达 63个。

1.4.2.2. UP 主模式—UP 主+虚拟人

虚拟 UP 主指以原创的虚拟角色设定、形象为自身的代表，在视频网站、社交平台上进行娱乐活动的艺人。典型代表是绊爱。虚拟 UP 主的活动形式非常自由且多样化， 每一位虚拟 UP主会探索并采取符合自身具体情况的活动形式。按照媒体分类，虚拟 UP主的活动有视频投稿、直播、图像、社交互动、线下活动等。按照具体活动内容，题材包括但不限于自我介绍、杂谈、Vlog、短剧、游戏实况、广播节目、音乐、演唱 会、动画等等，以及与其他虚拟 UP主或现实艺人的联动。虚拟 UP 模式主要流行于 日本和中国，目前主要活跃于 YouTube 和 Bilibili这两个平台。1）YouTube 是国外多 数虚拟UP主和事务所活跃的平台。在YouTube上活动的虚拟VTuber要想取得收益，需要申请收益化，通过后方可享有诸如 Super Chat 之类的收益方式。截至 2020 年 11 月，VTuber 总数已达到13000 人，比 2018 年增加了十倍。2）B 站是国内最大的二 次元虚拟人直播平台。虚拟主播进入国内后衍生出了 Vup模式，典型代表是 B 站。 B 站二次元属性较强，具有转 Vup 的观众基础和内容基础，用户粘性较大。同时，B站具有良好的社交机能，而且收益化门槛很低，主要靠直播打赏，竞争压力较小，观 众包容度高，对于个人 UP 主的发展比较友好，因此近年来UP 主模式快速发展。

1.4.2.3. 电商模式—主播+虚拟人

虚拟电商主播可帮助商家降低成本和对主播个体的依赖。虚拟电商主播也分为 AI 智能驱动和真人驱动，其中 AI 智能驱动型主播相比真人电商主播的优势最大，具体体 现在虚拟 AI 电商主播不受时间和精力的限制，可 24小时不间断带货，能够有效降 低成本，是替代人工的最佳选择。此外，长期来看虚拟电商主播的成本较低且稳定可靠，商家自有的虚拟电商主播不会因为流量变大而成本提高，从而使商家摆脱了对主 播个体的依赖。

虚拟电商主播和真人主播商业模式类似，差异体现在发展阶段和表现形式。1）虚拟电商主播和真人电商主播的平台和收入结构上类似。从直播平台来看，虚拟电商主播和真人电商主播的主战场都是淘宝、天猫和抖音；从收入结构来看，和真人主播类似，虚拟主播的收入也包含坑位费和销售提升，且不逊色于真人主播——洛天依淘宝直 播坑位费可达 90 万，而同期罗永浩的坑位费仅为 60万；洛天依首次直播在线观看 人数一度高达 270 万，近 200 万人打赏互动，位居当月直播观看人数排行榜第二名；2）虚拟电商主播形式多样，起步迅速。从展现形式来看，虚拟电商主播可采用偏真 人形象的虚拟 IP，也可以借助动画 IP形象，如“我是不白吃”，“一禅小和尚”，“猪 小屁”等，扩大了对应的消费者年龄范围；从带货品种来看，虚拟电商主播的风格偏二次元，用户以 Z 世代居多，销售的产品以休闲食品、二次元 IP 的周边衍生产品、少女服饰、潮牌、饮料等为主；从发展阶段来看，相比目前虚拟主播的收入规模仍在 起步阶段，但付费用户的同比增长率较高，可达 B 站水平的2 倍左右。3）品牌虚拟 电商主播是品牌量身打造的虚拟电商主播，如：2020 年 9 月欧莱雅推出了虚拟主播 “欧小蜜”后，在双 11预售阶段，欧小蜜的每天直播累计观赏人数达到 35 万以上， 带货总 GMV 也基本达到 170 万以上。

1.4.2.4. 代言模式—明星+虚拟人

原创虚拟 IP 人气较高后可跟真人一样做代言。具体来看有超写实 KOL 和虚拟歌姬、 品牌虚拟IP 和明星虚拟 IP 三大类。

1）超写实虚拟 KOL 非常接近真人 IP 的形象， 因此在商业模式上也最接近真人 IP的培养模式，人气积累到一定程度后可以像真人IP 一样做品牌代言。超写实虚拟 KOL“AYAYI”迅速积累人气，小红书首发帖阅读 量近300 万，一夜涨粉近 4 万。目前 AYAYI、翎 Ling 等超写实虚拟 KOL 代言费在百万元左右。高收入的背后是超高的制作成本，据称，超写实虚拟 KOL 视频每秒的 成本通常在 8 千至 1.5万元之间，一张图片也要数千元。

2）品牌虚拟 IP，部分公司 以自身品牌为核心打造虚拟IP，钉钉（钉哥、钉妹）、欧莱雅（欧爷、M 姐）、花西 子（花西子）、屈臣氏（屈晨曦wilson）等已开始尝试。品牌一般将虚拟人制作、IP 打造等全流程服务委托给虚拟人制作公司，虚拟人制作工作收取服务费用。部分本身具有雄厚技术基础的公司可自行打造虚拟人，如腾讯的无限王者团。

3）明星虚拟 IP， 是将真人明星的流量和热度扩大化。虚拟人制作公司为真人明显打造虚拟形象，从明星虚拟 IP 所获收益中分成。该种商业模式属于真人明星的衍生，并不属于独立的虚拟人。4）虚拟歌姬也可进行代言，但虚拟歌姬的出圈程度相对有限，且头部虚拟歌 姬较少，因此虚拟歌姬在代言领域布局有限。

1.5. 虚拟人产业链拆解—技术、运营、平台三角度划分

虚拟人全球产业链与国内产业链。1）全球来看，虚拟人产业链可分为上中下游三类。虚拟人上游包括内容制作类、工具类和 IP 策划类；中游主要指 XR 厂商、CG 厂商、 AI厂商和综合类厂商；下游主要涵盖企业服务、文娱领域和品牌代言。2）我国虚拟人产业链可按技术、运营和平台三个角度划分。我国虚拟人产业链上游工具类公司较少，中游专长类技术厂商和平台类垂直虚拟人厂商较多，下游内容运营市场庞大，因此可以将我国虚拟人产业链从技术、运营和平台三个角度进行划分。

1.5.1. 技术端—建模、驱动和渲染三大技术

建模技术分为静态扫描和动态扫描。建模包括静态扫描建模和动态光场重建。1）静态重建技术仍为主流，在影视行业有扎实的技术基础，其中相机阵列扫描重建逐渐替代结构光扫描重建成为静态扫描建模的主流方式，商业化程度较高；2）动态光场重建技术是目前世界上最新的深度扫描技术，可以忽略材质，直接采集世界的光线，然后实时渲染出真实的动态表演者模型，高品质重现不同视角下观看人体的光影效果，是虚拟人建模技术的发展方向。主要涉及计算机视觉（CV）和计算机图形学（CG）和计算摄像学技术。动态光场重建推荐关注在动态光场重建技术领域有较多积累的 龙头技术公司。

驱动技术包括动作捕捉驱动型和 AI 生成驱动型。1）动作驱动仍是主流，原理是将“中之人”的动作同步到虚拟人身上，具体包括惯性动捕、光学动捕和视觉动捕三类技术，其中光学动捕是对人物动作还原精度最高，但对于背景环境的要求和技术门槛 也较高，是未来动作驱动技术的发展方向；2）AI生成驱动型是让虚拟人具有自我学 习的能力，虚拟人的语音、表情、动作主要通过深度学习实时或离线生成驱动，而非 由中之人驱动，具体 AI技术包括基于文本实时生成虚拟人的语音和三维动画技术 （TTSA）、基于真人语音输入实时生成虚拟人三维动画技术（STA）、有感情的语音合成（ETTS）、对话系统（NLP）、智能动作生成、音色转换技术（VC）等。随着 AI驱动技术的发展，虚拟人制作的技术成本、互动性和对中之人的依赖性会逐渐降低， AI 生成驱动是虚拟人的发展方向，利好 AI技术扎实的技术型公司。

渲染技术分为离线渲染和实时渲染。1）离线渲染离线渲染涉及 PBR 技术和重光照技术，渲染时间长，质量高，计算资源丰富，大部分 CG 动画就是通过离线渲染呈现的，适合对精度要求高的虚拟人。2）实时渲染实时变化，每一帧都是针对当时实际的环境光源、相机位置和材质参数计算出来的图像，缺点是要受系统的负荷能力的限制，必要时会牺牲画面效果来满足实时系统的要求，更适合游戏或交互式场景。随着硬件能力和算法的提升，渲染速度、渲染效果、渲染画面的分辨率均大幅提升，同时未来虚拟人也将开拓虚拟社交新蓝图，因此实时渲染会是渲染技术的未来发展方向。

1.5.2. 运营端—IP 运营和 IP 资源两大类

IP 运营型公司是虚拟 IP 制作的核心。IP 运营型公司主要负责虚拟 IP 的孵化、形象设计、后期运营等，贯穿虚拟 IP 制作的始终，对于虚拟 IP 的最终变现能力至关重要。 IP 可以分为基于品牌/角色/真人创作的 IP和原创 IP，根据品牌/角色/真人创作的公司 需要有丰富的 IP 资源，娱乐、影视和出版类公司具有优势。相比之下，原创 IP 则要 求IP 运营型公司从 0 到 1 打造 IP 形象，培养周期较长，前期资金投入大，对 IP 运营公司的运营能力要求也更高，数字营销和传媒类公司具有优势。

IP 资源型公司在打造虚拟 IP 方面具有天然优势。IP 资源型公司具有丰富的原创影 视、文学IP 资源的公司。对于有较高角色魅力和粉丝基础的 IP 形象，IP 资源型公司 可通过虚拟人技术将这些 IP以更加立体、生动的方式展现出来，在获得人气积累后 可以尝试电商模式、代言模式等，实现出圈。

1.5.3. 平台端—制作平台和内容平台两大类

虚拟人制作平台的产生可以推动消费级 AI 虚拟助手的落地。虚拟人平台可以帮助企业节约人力成本、提升效率、扩大经济效益，推动虚拟人应用向平台化方向发展。弊端在于虚拟人形象可能存在同质化问题，消费者对同质化的虚拟人形象接受度可能不高。未来，能够根据企业产品和所在行业特点进一步定制化产出高性价比虚拟人是 平台模式的发展方向。

内容运营平台型公司为 2C 端虚拟 IP 提供运营平台。内容运营平台型公司为虚拟 UP主和虚拟电商主播提供和用户沟通的平台，属于 UGC 平台层面的运营，对平台内虚 拟 IP数量和用户粘性有较高要求，典型代表是哔哩哔哩和快手。（报告来源：未来智库）

2. 虚拟人板块发展驱动力—政策端、产业链端和需求端

2.1. 政策端—符合政策导向

虚拟人符合数字经济发展规划。近年来国家多次出台政策鼓励数字经济相关行业和 技术的发展。2022年 1 月，国务院公开发布《\”十四五”数字经济发展规划》，指明数 字化转型是我国科技革命和产业变革的发展方向，发展目标是到 2025年数字经济迈 向全面扩展期，数字经济核心产业增加值占 GDP 比重达到 10%，智能化水平明显增强，数字技术与实体经济融合取得显著成效。发展规划所鼓励的多个数字化场景囊括虚拟人所有商业模式的应用，包括社交、购物、娱乐、电商、展览、智慧导览、智能导流、虚实交互体验等。作为元宇宙数字经济的行为主体，虚拟人可应用到数字身份、虚拟房地产、数字劳动、数字消费、社交互动等多个领域。

虚拟人直接受益于传媒行业的数字化转型。2021 年 10 月，国家广播电视总局发布《广播电视和网络视听“十四五”科技发展规划》，明确指出鼓励虚拟主播、动画手语广泛应用于新闻播报、天气预报、综艺科教等节目生产，创新节目形态，提高制播 效率和智能化水平。到 2025 年，IP化制播标准体系基本完善；各级广播电视播出机 构制播系统广泛实现 IP 化，制播流程更加优化，内容生产智能化水平显著提升。同时探索短视频主播、数字网红、直播带货等虚拟形象在节目互动环节中的应用，增加个性化和趣味性。该发展规划显示了国家对虚拟人的应用以实用性为主，以娱乐性为 辅的政策导向。

2.2. 产业链端—硬件和技术和核心驱动力

虚拟人与手游产业有一定的可类比性，手游板块行情发展因素可做参考。通过对比产业链结构，我们发现虚拟人可类比手游行业前期的发展，两者都对于 IP 内容的打造有着较高的要求，同时要求技术上的跟进与消费者教育的进步，手游板块行情发 展因素可做参考。以长江游戏指数相对于万得全 A指数的相对比值来衡量游戏行业 的行情始末时点。时间段上，考虑到相当一部分游戏公司在创业板上市，因此我们 参考 2009 年 10 月30 日创业板开板以后的 A 股行情表现。

手游板块产业链上游崛起带动手游爆发。1）手机提供硬件支持，期间游戏行业迎来了其历史上爆发性最强的一轮行情，其所处的 2013 年是手游爆发的元年，2013 年， 智能机在迅速普及下，渗透率有大幅提高，根据2013 年 7 月 Google 的智能手机使 用情况调查报告，中国城市智能手机的普及率大大提高，从 2012 年的33%上升至47%，近一半的中国城市居民拥有智能手机。2）安卓系统提供技术支持。2007 年 11 月 Google发布了安卓的源代码。2008 年 10 月，第一部安卓智能手机发布。之后安卓逐渐扩展到平板电脑及其他领域上，如电视、数码相机、游戏机、智能手表等。 2011年第一季度，安卓在全球的市场份额首次超过塞班系统，跃居全球第一。 2013 年的第四季度，安卓平台手机的全球市场份额已经达到78.1%。

2.3. 需求端—价值创造推动需求不断提升

虚拟人的需求本质上取决于其为社会创造的价值。虚拟人的本质仍然是数字化技术所带来的对现实人工的替代，其对社会的积极意义根据虚拟人类别可以从服务型虚 拟人和身份型虚拟人两个角度阐述。

服务型虚拟人可实现降本增效，替代某些机械性、重复性、说明性、定制化的人力工作。1）替代标准化程度较高的工作内容，降本增效。比如说对于银行的柜员、餐厅店员、政务服务人员、新闻主播等等，一方面用可直接复制的标准化的虚拟人服务模式可以节省大量的人力成本（初始投入费用相对巨大，但是后期仅需较低的维护费用），另一方面可以提高服务效率，虚拟人有着设定好的程序执行步骤，出错概率极小，最后同时也可以提供一致的服务态度，给客户的体验感将会更好。2）替代陪护型的工作，特别对于部分生活陪伴、顾问等耗时较高，针对性较强的投入，虚拟人的应用可以更加长期、定制化的服务于客户，成为客户工作生活的助手，提高效率。随着人工智能、深度学习等技术的发展，服务型虚拟人的智能程度提升，将 囊括更多交互场景和内容，使虚拟更“真实”。

身份型虚拟人相对真人 IP 优势明显，未来有望开拓元宇宙“第二分身”。身份性虚拟人的存在更多强调其 IP 属性，IP 将是虚拟人、虚拟分身打造的核心，也是建立其粉丝经济的核心，而身份型虚拟人的建立，积极意义有：1）减少真人给 MCN 机构、 营销主体等的风险。传统的 IP流量中心基本都是真人，无论是明星还是网红主播， 作为相对独立的个体，虽然前期会与其服务的 MCN 机构或代言的厂家有一定的约束合作，但后两者仍面临诸多不确定性，比如可能会因为合作对象的个人问题影响到企业的品牌形象从而带来巨额损失等，但虚拟人的应用由于其由数字特性，任何对外活动均由运营方提前设置而来，减少了个人因素的风险。2）成本可控、风格可变、长效价值。虚拟人作为运营方的一种虚拟资产，不再是之前的厂商与明星/主播之间的合作渠道关系，不用担心品牌价值与营销渠道之间的过度绑定性，虚拟人不存在涨价、跳槽等方面的潜在风险，同时由于其可塑性强，可以跟随粉丝的喜好而进行改变其风格，也不会随时间衰老，具有更加长效可控的资产价值。3）个人身份型虚拟人就是未来元宇宙世界的第一形象。随着元宇宙相关技术的成熟，能够在元宇宙内进行的活动越来越多也越来越沉浸时，每个人都将在虚拟世界内有自己的一 个虚拟分身，每个人都是自身独一无二的IP，虚拟分身作为人类在元宇宙世界内的 第一形象，将在未来的广告、直播、社交方面有更多的应用。

虚拟人市场规模将快速扩大。在 2030 年，我国虚拟数字人整体市场规模将达到 2700亿。其中，虚拟 IP 发展潜力巨大，主要得益于我国巨大的虚拟人下游需求市场，虚拟人在传媒、二次元、电商、音乐、广播、金融等多个领域的都有广泛的应用。身份 型虚拟数字人将占据主导地位，约 1750 亿，并逐步成为Metaverse 中的重要一环。 服务型虚拟数字人则相对稳定发展，多模态 Al 助手仍有待进一步发展，多种对话式服务升级至虚拟数字人形态，总规模超过 950 亿。同时，虚拟人带动市场规模增速同 样可观，根据艾媒咨询预测，2021 年中国虚拟 IP带动市场规模为 1075 亿元，到 2023 年该数字有望达到 3335 亿元，2021-2023 年的符合增长率达到76.13％。

3. 虚拟人板块当前存在哪些投资机会？

3.1.受益于政策推动的国资背景公司—更好承载虚拟人数字经济属性

3.2.布局产业链上游前沿技术的公司—硬件/技术类公司占优

全球范围来看，当前虚拟人行业具备此前驱动游戏板块占优的硬件/技术基础条件。游戏行情崛起主要受到硬件/技术相对成熟之后带来的应用端爆发。而当前虚拟人行业面临着类似的驱动力，1）硬件方面，头显、OLED、透明屏、电脑、投影屏、全息屏、动作捕捉系统等基础硬件设备已经发展的比较成熟，同时，消费级的 VR 设 备 Quest2已经推出，有望释放虚拟人在虚拟分身方面的发展潜力。2）技术方面， 当前技术已较为成熟，建模、驱动、渲染、AI等底层技术已经可以使虚拟人的仿真 程度和互动性达到较高水平。全球范围来看，虚拟人产业链上游技术端基本完成搭建，产业链中游专场类技术厂商蓬勃发展。全球范围来看，硬件和技术条件已基本满足，虚拟人产业链逐渐打通，商业模式得到验证，虚拟人的价值得到体现。

目前我国虚拟人行业受制于技术，发展需求较高，利好前沿技术公司。全球虚拟人产业链打通的背景下，我国虚拟人行业却并未见到爆发式增长，主要是因为我国产业链上游技术薄弱，还没有形成统一的、标准化的算法模型，生产模式和规格仍在探索中，并且国内还没有全栈式垂直虚拟人厂商。技术的进步可带来下游需求的加倍释放，因此发展上游技术的需求较高。考虑到和国外虚拟人制作水平的差距，我国虚拟人技术 端公司还有较大发展空间。

整体来看产业链上游技术方面发展空间较大。同时，受到产业链下游需求的推动，产业链上游的边际效益提高，受到资本和市场的关注度逐渐提高。结合上游技术的发展状况和趋势，我们认为在 动态光场重建、光学捕捉、AI生成驱动和实时渲染等方向有良好基础的公司具有较 大发展潜力。

3.3.技术限制减少后需求爆发的公司—关注恐怖谷两侧的投资机会

技术进步在预期之内，长远来看需求爆发是行业大势。虽然我国目前虚拟人制作技术相较国内还有较大差距，但技术方面不存在卡脖子问题，国内追赶国内技术水平只是时间问题。长远来看，随着产业链上游技术的进步，产业链下游的应用需求会加倍放 大，因此需求爆发的方向即是虚拟人的投资机会。

上游技术进步和技术成本下降将推动虚拟人向恐怖谷两侧发展。恐怖谷理论是指随着类人物体的拟人程度增加，人类对它的好感度亦随之改变。在“接近人类”程度的时候，人类好感度会突然下降。目前整体技术水平已接近恐怖谷，未来，我们预计在技术成本未全面的、显著的降低之前，虚拟人会朝两个方向发展：一类止步于当前的技术水平，开始往运营方向发展，止步于恐怖谷之前；另一类随着技术的提高而应用升级，不断提高虚拟人的仿真程度和智能程度，尝试跨越恐怖谷。

3.3.1.止步于恐怖谷之前—进入用户数量快速增长阶段

止步于恐怖谷之前有红海市场和蓝海市场两个发展方向，主要围绕虚拟 IP 展开，利好我国虚拟人产业链中 IP 运营、制作平台和 IP 资源型公司。

3.3.1.1. 虚拟 UP 主人和虚拟电商—红海市场

虚拟 UP 主和虚拟电商模式是红海市场，技术成本下降将促使市场参与者提高服务质量。红海市场的典型市场是虚拟 UP 主和虚拟电商。对于虚拟 UP 主，用户是出于 对可互动的二次元形象的喜爱才喜欢上虚拟 UP主，技术成本下降并不会提高虚拟 UP 主的写实程度，但会扩大虚拟 UP 主的数量，技术成本因素对虚拟 UP 主模式发展的影响将越来越小。对于虚拟电商，其核心在于互动而不在于主播的仿真程度，因此技术成本的下降会推动虚拟电商行业横向发展，数量快速增加，而非在仿真度上不 断精进。

深耕垂直领域和降低平均制作成本是提高虚拟人红海市场的关键，利好 IP 运营型和制作平台型公司。1）深耕垂直领域方面。与传统 UP 主和电商类似，随着平台竞争加剧，内容产品同质化现象愈来愈重，追求差异化才能脱颖而出。因此虚拟 UP 主人 和虚拟电商也需要打造细分领域的垂直IP。以一禅小和尚为例，其核心是暖萌和治 愈，唯美的中国风格的背景，一禅小和尚关于情感、成长、孤独的发问和趣事，戳中了当代青年内心隐秘的情感痛点，而虚拟电商直播可以灵活互动的方式把这种情感价值转变成商业价值；2）降低制作成本。虚拟人制作成本降低也是虚拟人红海市场发展的关键。目前虚拟人直播间价格仍较高，蓝色光标与阿里巴巴达摩院共同打造的虚拟直播间价格为 9.9 万元/间，目前只有部分大品牌能够承担起虚拟电商的成本，小品牌受制于成本无力享受虚拟主播的便利。考虑到电商直播收入头部集中度较高，未来虚拟直播间价格的降低带来的品牌的需求提升将是指数级的，具有广阔的市场 空间。利好虚拟人产业链中运营端的 IP运营型和制作平台型公司。

3.3.1.2. 虚拟 IP 立体化—蓝海市场

IP 立体化是蓝海市场，发掘潜在顾客和创造新的价值。IP 立体化是将漫画、影视、 文学中的IP 形象以虚拟人的形式展示出来，通常要经历文字-动画-虚拟人的发展顺 序，重点在于将人们喜爱的 IP以能够与用户互动、交流的方式展现出来，而非在于 虚拟 IP 的仿真程度，因此属于止步于恐怖谷的范畴。技术成本降低后将产生巨大的市场需求，存在巨大的市场空间，可通过多种虚拟人商业模式变现。典型代表是阅文 集团旗下的虚拟 IP叶修。叶修承接电商、代言等多种商业模式。叶修代言的商品涵 盖服饰、饮料、耳机、洗面奶、快餐等，还在上海和杭州打造了《全职高手》麦当劳主题店。
